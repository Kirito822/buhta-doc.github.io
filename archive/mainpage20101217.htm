Отчеты

 

[ * ] [17233]

В связи с изменениями, произведенными в релизе 111 (см. заявку [17214]) для отчетов с настройкой показателей (доступность полей настройки Список субсчетов и Список корр.счетов), внесены дополнительные изменения в запросы для следующих форм отчетности:

- Сведения о производстве и отгрузке товаров и услуг, форма № П-1.

- Сведения об объеме платных услуг населению по видам, приложение № 3 к форме № П-1

- Сведения о состоянии расчетов за отгруженную продукцию, выполненные работы (услуги) - Форма № 1-РП (срочная)

- Сведения о финансовом состоянии организации, форма № П-3

- Основные сведения о деятельности организации, форма № 1-предприятие

- Сведения о затратах на производство и продажу продукции (товаров, работ, услуг), форма № 5-З

- Сведения об основных показателях деятельности малого предприятия, форма № ПМ

- Сведения о движении ценных бумаг и доходах по ним по институциональным секторам, форма № 1-ЦБ

- Приложение к бухгалтерскому балансу

Добавлены также новые процедуры, внесены изменения в формы настройки.

Все измененные объекты входят в состав стандартной конфигурации BuhtaForm.mdb

 

Кадры

 

[ * ] [17354]

Внесены следующие исправления в режим работы Табеля учета рабочего времени:

- при нажатии на кнопку Печать не производится автоматического сохранения  в таблицу Табель изменений, сделанных в экранной форме, вместо этого  запускается процедура расчета количества дней-часов и неявок по экранной таблице, что позволяет учитывать при печати сделанные в экранной форме изменения.

- при нажатии на кнопку Сохранить из таблицы Табель удаляются все записи, которые соответствуют указанному на экране фильтру за указанный период, после чего в таблицу Табель добавляются записи с экранной формы.

 

Настройка

 

[ * ] [17290]

В связи с конфликтом наименований полей в окне со списком запросов (режим Настройка – Инструменты - Запросы) произведены следующие изменения пользовательского интерфейса:

- переименованы столбец Группа в списке запросов, поле Группа для указания фильтра в окне со списком запросов, поле Группа в карточке редактирования запроса - новое название для этих данных Раздел;

- в список запросов добавлен новый столбец Группа, в котором отображается значение поля Группа из конфигурации.
