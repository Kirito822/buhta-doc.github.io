Отчеты

 

[ - ] [20395]

При вызове режима построения Сводного отчета из АРМ-а заголовок окна, содержащий имя отчета и период, не обновлялся при их замене. Ошибка исправлена.

 

Списки

 

[ + ] [20493]

Разработан режим фильтрации списков из буфера обмена. Вызывается из контекстного меню списка, пункт Фильтр - Фильтр из буфера обмена. В результате применения этого фильтра в списке остаются только те строки, у которых значение текущего столбца (текстовое) присутствует в списке значений буфера обмена.

 

Информация для разработчиков

 

[ * ] [20496]

Исправлена процедура DialogSQLStrokaWithCheck -  выставление отметок у записей списка для последующего отбора. Если  в списке присутствует значение <Все>, то при его выборе снимается отметка с других отмеченных строк и наоборот, если помечается любая строка кроме  <Все>, снимается отметка со строки  <Все>.

 

[ * ] [20506]

Внесены следующие доработки в Lookup-форму:

1. Добавлен подсчет выбранных записей в SelectButton в режиме отметки записей.

2. Добавлено свойство LookupForm.NoFocusedToResult

3. Разработана функция DoShowSelectLookupFormWithEmptyResList, позволяющая возвращать пустой список при выборе.

 

Зарплата

 

[ + ] [20463]

В режиме сбора данных о начисленных и уплаченных взносах на ОПС (вызов через пользовательскую форму [З/П. Экспорт в XML-файл для ПФР. Формат 2014г]) добавлена возможность создания, печати и выгрузки Инд.сведений с типом СЗВ-К. В результате модернизации добавлены новые поля в таблицы [ПФР Инд.сведения] и [ПФР Стаж], вследствие чего требуется обновление структуры базы данных эталоном текущей версии.

См. также соответствующий раздел справки Конфигурация ''Стандартная_Зарплата_...''.
