[ * ] 18675

•В соответствии с Приказом ФНС РФ от 23 января 2012 г. N ММВ-7-3/13@ добавлена новая форма Налоговой декларации по единому налогу на вмененный доход для отдельных видов деятельности.

Отчет заполняется по новому шаблону [Стандартный. Налоговая декларация по ЕНВД 2012] начиная с 1 квартала 2012 года.

Пользователю следует в новом запросе [Стандартный. Налоговая декларация по ЕНВД (2012)] - используется при построении отчета начиная с периодов 2012 года - настроить алгоритмы расчета для строк 030 и 040 Раздела 3 .

•Добавлена выгрузка в файлы XML отчета составленного по шаблону [Стандартный. Налоговая декларация по ЕНВД 2012].

 

Все измененные и добавленные объекты, связанные с указанной модернизацией, входят в состав конфигурации BuhtaForm.mdb.

 

[ * ] 18676

•В соответствии с приказом ФНС РФ от 20.02.2012 № ММВ-7-11/99@ добавлена новая форма Налоговой декларации по транспортному налогу. Отчет заполняется по новому шаблону [Стандартный. Налоговая декл. по трансп.налогу 2012] начиная с 2012 года.

Требуется настроить новый запрос [Стандартный. Налоговая декл по трансп. налогу 2012], по которому составляется отчет. В запросе следует указать коды и суммы льгот. Если код заполнен, то в отчет выводятся остальные данные об этой льготе.

 

Соответствия полей запроса и кодов строк - льгот в отчете:

150 - [ЛьготаОсвоб КолМес]

160 - [ЛьготаОсвоб Коэффициент]

170 - [ЛьготаОсвоб Код] / [ЛьготаОсвоб Основание]

180 - [ЛьготаОсвоб Значение]

190 - [ЛьготаСумма Код] / [ЛьготаСумма Основание]

200 - [ЛьготаСумма Значение]

210 - [ЛьготаСтавка Код] / [ЛьготаСтавка Основание]

220 - [ЛьготаСтавка Значение]

Нельзя одновременно заполнять льготу в строке 170, и в строке 190 и 210, либо только 170, либо 190 и/или 210

 

•Добавлена выгрузка в файлы XML отчета составленного по шаблону [Стандартный. Налоговая декл. по трансп.налогу 2012].

 

Все измененные и добавленные объекты, связанные с указанной модернизацией, входят в состав конфигурации BuhtaForm.mdb.

Изменены процедуры работы с семафором:

 

SetSemaforOn - процедура добавляет слева на Главную инструментальную панель кнопку с изображением мигающего фонаря -

добавлены указание периода мерцания в мсек (значение по умолчанию 1000)  и цвет (значение по умолчанию зеленый)

 

procedure SetSemaforOn(const FlashPeriod: integer=1000; const Color: TColor=clYellow);

добавлены указание периода мерцания в мсек (значение по умолчанию 1000), цвет (значение по умолчанию зеленый), положение левого верхнего угла формы семафора (по умолчанию - в центр экрана).

 

procedure ShowSemafor(const FlashPeriod: integer=1000; const Color: TColor=clYellow;

const X: integer=-1; const Y: integer=-1);

[ + ] 18606

В режим сканирования (контекстное меню стандартного списка, пункт Сервис – Сканировать) добавлена возможность записи многостраничного документа в один файл формата jpg. При этом отсканированные листы склеиваются слева направо.

 

[ + ] 18685, 18709

•Доработаны режимы Поиск и Поиск далее из контекстного меню рабочего стола. Теперь результатом поиска являются не только ярлыки АРМ-а, но и папки рабочего стола, в которые входит искомый ярлык. Во вложенных папках ярлыки в этих режимах не ищутся.

•В контекстное меню Папки рабочего стола добавлено два пункта Поиск и Поиск далее, аналог режимов поиска на рабочем столе.

•В контекстное меню рабочего стола добавлен пункт Поиск в папках. Результатом поиска является список ярлыков АРМ-а (название которых включает указанную комбинацию символов) с указанием всех папок рабочего стола, в том числе и вложенных, в которых находится искомый ярлык. По двойному нажатию мышки на записи соответствующая папка открывается и курсор устанавливаться на нужный ярлык.
