Отчеты

 

[ + ] [19323]

Добавлены новые отчетные формы в рамках бухгалтерской отчетности для малых предприятий. Все объекты, добавленные и измененные в рамках этой модернизации, входят в стандартную конфигурацию BuhtaForm.

Алгоритм настройки показателей и составления бухгалтерской отчетности не отличается от алгоритма, используемого ранее при работе с пользовательской формой [Стандартная. Бухгалтерский баланс 2011], с помощью которой производится настройка и печать стандартных бухгалтерских отчетов.

Работа с бухгалтерской отчетностью для малых предприятий осуществляется через пользовательскую форму [Стандартная. Бух.отчетность для малых предприятий], по шаблону [Стандартный. Бух.отчетность для малых предпр. 2013].

 

В поставку текущей версии входит файл Пример_настройки_ФО.mdb, который содержит образец настройки для отчетов Бухгалтерский баланс (код формы Ф1_2013м), Отчет о финансовых результатах (код формы Ф2_2013м), Отчет о движении денежных средств (код формы Ф4_2013м). Пользователю следует воспользоваться формой [Импорт настроек ФО] и сымпортировать примерные настройки новых показателей, после чего их отредактировать.

Обращаем внимание, что для Отчета о движении денежных средств для малых предприятий запрос полностью повторяет запрос для составления этого отчета в стандартной бухгалтерской отчетности, т.е. его настройку можно скопировать из запросов [Стандартный. Форма Ф4_2012/1] и [Стандартный. Форма Ф4_2012/2]. Поэтому для этого отчета при нажатии на кнопку <Настройка запроса>  проверяется наличие настройки запроса  [Стандартный. Форма Ф4_2013м/1] (для текущего периода) или [Стандартный. Форма Ф4_2013м/2] (для предыдущего периода) (т.е. наличие записей в таблице [_Настройка форм отчетности]). Если таких настроек нет, но есть настройки для запроса [Стандартный. Форма Ф4_2012/1] ([Стандартный. Форма Ф4_2012/2]), то на экране появляется сообщение с подтверждением на копирование этих настроек в новый запрос.

 

Добавлена выгрузка отчета составленного по шаблону [Стандартный. Бух.отчетность для малых предпр. 2013] в файл формата XML - исправлены формы [Стандартная. Архивный отчет в эл.виде (Дополнения)] и [Стандартная. Архивный отчет в электронном виде].

 

[ + ] [19934]

В стандартную конфигурацию Унифицированные формы (файл BuhtaUniform.mdb), группа Счета-Фактуры добавлены шаблоны и запросы для составления Универсального передаточного документа (УПД)  - отчета, который объединяет в один документ счет-фактуру и товарную накладную  на отгрузку товаров.

Отчеты составляются в режиме Печать по шаблону, который вызывается по "горячей" клавише F10 из списка договоров/документов.

Из списка договоров доступны шаблоны:

- [Универсальный передаточный документ/Договор],

- [Универсальный передаточный документ/Дог/ГТД в спец].

Запросы к ним: [Печать.Универсальный передаточный документ.Договор], [Печать.Универсальный передаточный документ.Догспец], [Печать.Универсальный передаточный док.Догспец сГТД].

 

Из списка документов доступны шаблоны:

- [Универсальный передаточный документ/Документ],

- [Универсальный передаточный документ/Док/ГТД в спец].

Запросы к ним: [Печать.Универсальный передаточный док. Документ], [Печать.Универсальный передаточный документ.Докспец], [Печать.Универсальный передаточный док.Докспец сГТД].

 

Дополнительно исправлены грамматические ошибки в следующих шаблонах этой же группы Счета-Фактуры:

- [Счет-фактура 2012 на аванс],

- [Счет-фактура 2012 на реализацию (ГТД в спец-и)],

- [Счет-фактура 2012 на реализацию (док ГТД в спец-и)],

- [Счет-фактура 2012 на реализацию товара],

- [Счет-фактура 2012 на реализацию товара (дог)].

 

[ * ] [20039]

В соответствии с Приказом ФНС России от 14.11.2013 № ММВ-7-3/501@, который заменил Общероссийский классификатор объектов административно-территориального деления (ОКАТО) на Общероссийский классификатор территорий муниципальных образований (ОКТМО), внесены следующие изменения:

•В таблицы Юр.лицо и Подразделение и в соответствующие карточки редактирования добавлено новое поле ОКТМО

•Начиная с 4 квартала 2013 года в следующих отчетах

- Налоговая декларация по НДС,

- Налоговая декларация по транспортному налогу,

- Налоговая декларация по ЕНВД

вместо кода ОКАТО распечатывается код ОКТМО, если  поле ОКТМО заполнено.

•Изменена выгрузка указанных отчетов в файлы XML:

- Для Декларации по НДС начиная с отчетов за 4 квартал 2013 г. (или декабря 2013 г.) версия формата изменена на 5.03,

- Для Декларации по ЕНВД начиная с отчетов за 4 квартал 2013г. версия формата изменена на 5.04, вместо ОКАТО выгружается ОКТМО,

- Для Декларации по транспортному налогу начиная с отчетов за 2013 г. версия формата изменена на 5.02, вместо ОКАТО выгружается ОКТМО.

 

Модернизация требует обновления структуры базы эталоном текущей версии и импорта файла конфигурации BuhtaForm.

 

Зарплата см. также Конфигурация ''Стандартная_Зарплата_...''

 

[ * ] [20025]

Внесены изменения в алгоритм расчета страховых взносов в 2014 году в соответствии с ФЗ № 333-ФЗ от 2 декабря 2013 года:

- Отменен расчет накопительной части

- Действие пониженных тарифов, установленных частью 3.4 статьи 58 ФЗ 212-ФЗ, продлевается до 2018 года (тарифы для определенных категорий на УСН, аптечных организаций, ИП применяющих патентную систему, социально ориентированных НКО, благотворительных организаций)

- Действие основного тарифа (который должен был действовать по 2015 год) продлен до 2016 года.

Модернизация требует синхронизации базы данных эталоном текущей версии для обновления процедуры [dbo].[ЕСН Заполнение шкалы].

В следующие режимы добавлена возможность копирования значений строки / ячейки в буфер обмена:

- Оборотная ведомость

- Сводный отчет

- Карточка движения по субконто

- Карточка движения по счету

- Прайс лист

Режим копирования вызывается из контекстного меню списка,

пункт   Копировать в буфер обмена ----> Строку

                                         Ячейку

 

[ - ] [19961]

В режим просмотра объектов сеанса - недавно редактируемых объектов, с которыми работал пользователь (вызов осуществляется по "горячим клавишам" Ctrl+H или через Главную инструментальную панель) добавлена обработка настройки на тип карточки редактирования. Ранее в этом режиме при выборе какого-либо объекта сеанса вызывалась карточка редактирования по умолчанию, теперь -  та, которая указана в настройке вида соответствующего выбранному объекту.
