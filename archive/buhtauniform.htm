ГРУППА ОТЧЕТОВ ОС-1, ОС-1а, ОС-1б

Версия: 1.52

Описание:
Формы NN ОС-1, ОС-1а, ОС-1б используются для отражения различных операций, связанных с движением основных средств, в частности, для оформления операций:
- ввода в эксплуатацию ОС, поступивших посредством приобретения за плату;
- получения от других организаций по договору;
- продажи или передачи другой организации.

Поэтому для каждой из форм ОС-1, ОС-1а, ОС-1б разработано по три различных шаблона:"Акт ввода", "Прием" и "Передача".
Все отчеты составляются из списка документов.

- Отчеты "Акт ввода" составляются по проводкам, содержащим в поле "Дебет" счет учета ОС (01) и в поле "Дб тип субконто 2" значение "ОС".
- Отчеты "Прием" составляются по проводкам, содержащим в поле "Дебет" счет 08 и в поле "Дб тип субконто 1" значение "ОС".
- Отчеты "Передача" составляются по проводкам, содержащим в поле "Дебет" счет 01, в поле "Кредит" также счет 01(субсчет "Выбытие ОС"), и в поле "Кр тип субконто 2" значение "ОС".

Для форм ОС-1 и ОС-1а по каждому ОС формируется отдельный акт.
Для всех отчетов ОС-1 и ОС-1б используется одна форма запроса параметров - [Стандартная. ОС-1]. Функциональность формы соответствует типу создаваемого отчета (отключаются те или иные функционалы)
Для отчетов ОС-1а используется форма [Стандартная. ОС-1а]

ГРУППА ОТЧЕТОВ ОС-6, ОС-6а, ОС-6б

Версия: 1.58

Описание:
Форма ОС-6 - "Инвентарная карточка учета объекта ОС" печатается из списка ОС.
Раздел 1 формы ОС-6 ДОЛЖЕН ЗАПОЛНЯТЬСЯ ТОЛЬКО для ОС, бывших в употреблении и переданных из другой организации или самостоятельного подразделения, документ о вводе в эксплуатацию, фактический срок эксплуатации и сумма начисленной амортизации - все это данные с ПРЕЖНЕГО места нахождения. Предлагаемый шаблон предназначен для ОС, введенных в эксплуатацию в данной организации, поэтому раздел 1 не заполняется.
Форма ОС-6а - "Инвентарная карточка группового учета объекта ОС" печатается из списка ОС.
Форма ОС-6б - "ИНВЕНТАРНАЯ КНИГА учета объектов основных средств" печатается из формы [Стандартная. ОС-6б].

Для корректной работы и заполнения всех полей на документ накладываются следующие условия:
1. для ОС-6 - ОС на втором субконто кредита проводок с дебетовым или кредитовым субсчетом 01 счета
2. для ОС-6а - ОС на втором субконто кредита проводок с дебетовым субсчетом 01 счета

ОТЧЕТ ИНВ-11

Версия: 2.120

Описание:
Унифицированная форма N ИНВ-11 утверждена постановлением Госкомстата РФ от 18 августа 1998 г. N 88.

Применяется при инвентаризации расходов будущих периодов.

 

В форме запроса параметров заводятся:

- номер и дата документа,

- вид основания для проведения инвентаризации, его номер и дата,

- дата инвентаризации,

- счет (97 или субсчет 97-го),

- вид договора и его признаки (начало и окончание погашения расходов),

- срок погашения по умолчанию (для позиций, где списание расходов не оформляется договором)

- информация о членах комиссии и мат.ответственных лицах (ФИО, должности)

 

В версии 2.120 производится заполнение следующих полей отчетной формы:

шапка:

- Название организации по текущему Юр.лицу,

- Код ОКПО по текущему Юр.лицу,

- Структурное подразделение – по подразделению первого из материально ответственных лиц;

колонки:

- Вид расходов - наименование и код субконто (если есть субконто 2, расшифровывается по субконто 2)

- Общая сумма - сумма последней проводки по дебету указанного счета по соответствующей аналитике

- Дата возникновения - дата последнего дебета

- Срок погашения в месяцах -.из признаков договора – от даты начала погашения до даты окончания погашения, если нет договора - берется из формы ввода параметров

- Расчетная сумма = [Общая сумма / Кол.месяцев]

- Списано - сумма по кредиту указанного счета от даты возникновения до даты инвентаризации

- Остаток расходов = [Общая сумма – Списано]

- Кол.месяцев - от даты возникновения до даты инвентаризации

- Подлежит списанию - не заполняется

- Расчетный остаток - не заполняется

- Результаты инвентаризации - не заполняется

Сведения о численности и потребности организаций в работниках по профессиональным группам

Версия: 1.87

Описание:
Приказом от 07.07.2008 N 156 утверждена форма N 1-Т (проф) "Сведения о численности и потребности организаций в работниках по профессиональным группам". Представлять данную форму обязаны юридические лица (кроме субъектов малого предпринимательства), осуществляющие все виды экономической деятельности (кроме государственного управления и обеспечения военной безопасности, а также деятельности общественных объединений). Организации должны заполнять и сдавать форму 1 раз в два года, она вводится в действие, начиная с отчета по состоянию на 31.10.2008 (срок сдачи - 28 ноября).

 

При формировании отчета необходимо, чтобы должности в таблице "Кадры Должность" содержали ссылки на справочник ОКЗ (Общероссийский классификатор занятий).

 

В отчете по каждой строке в колонки 4 и 5 попадают данные по ЗП Статусу (см. описание запросов), из кол-ва по штатному расписанию вычитается кол-во по ЗП статусу, если полученное число больше 0, оно попадает в колонку 6.
