[ + ] [17738]

Добавлена возможность при составлении Платежных поручений по подоходному налогу и страховым взносам создавать спецификации документов по каждому сотруднику. В связи с этой модернизацией в программу внесены следующие изменения:

•Изменена структура таблицы [ЗП Перечисление] - добавлены поля [ПП по сотр] - признак необходимости создавать докспец по сотрудникам, [ПП по сотр Вид докспец] - вид докспец, которую надо создавать по сотрудникам, [ПП по сотр По докспец] - вид докспец, по которой собираются данные о начислении сотрудникам. Требуется обновление структуры базы данных эталоном текущего релиза.

•Изменена карточка редактирования ЗП Перечисление (меню Зарплата - Настройка - Перечисление, кнопки <Добавить>, <Изменить>):

- Для перечисления с типом Подоходный налог добавлен флаг Создавать докспец с суммами перечислений по каждому сотруднику. Если он выставлен, то становится видимым параметр Вид докспец Перечисление по сотруднику для указания вида докспец, которую надо создавать.

- Для перечисления с типом Единый социальный налог добавлен флаг Создавать докспец с суммами перечислений по каждому сотруднику. Если он выставлен, то становятся видимыми параметр Вид докспец Перечисление по сотруднику для указания вида докспец, которую надо создавать, и параметр Суммы к перечислению искать по виду докспец -  для указания вида докспец, по которому следует собирать данные о начислении по сотрудникам (в последнем случае показываются виды спецификаций для вида документа, определенного параметром Вид документа С/В в Основной настройке ЗП на закладке Страх.взносы).

•В Основную настройку ЗП (режим Зарплата - Настройка - Основная настройка)

- на закладку Подоходный добавлен параметр Виды докспец-перечислений налога по сотрудникам для указания видов докспец, по которым собирать данные о перечислении по сотруднику (на список видов докспец наложен фильтр - в список попадают только виды докспец, которые указаны в настройке карточек ЗП Перечисление в качестве видов докспец, которые создаются как перечисление по сотруднику); будет использоваться при составлении справки 2-НДФЛ и Налоговой карточки, а также - при экспорте в ГНИ данных о доходе и подоходном налоге.

- на закладку Страх.взносы добавлены параметры для указания видов докспец,  по которым собираются перечисления ОПС (страх. и накоп. частей) по сотрудникам (наложен тот же фильтр  - в список попадают только виды докспец, которые указаны в настройке карточек ЗП Перечисление в качестве видов докспец, которые создаются как перечисление по сотруднику); будут использоваться при экспорте в ПФР данных об ОПС.

•Исправлена пользовательская форма [Стандартная. Составление платежных поручений] (вызывается также из режима Зарплата - Составление платежных поручений). Входит в состав конфигурации BuhtaConfig.

 

Алгоритм составления платежных поручений:

 

Для перечисления с типом Подоходный налог данные собираются по проводкам того вида, номер которых совпадает с номером шифра, указанного в настройке ЗП Перечисления. Спецификации по сотрудникам при этом создаются так:

- в субконто 1 заносится Сотрудник - берется из той записи таблицы Зарплата, на которую ссылается проводка,

- в субконто 2 заносится Расчетный месяц (значение, которое указывается в соответствующем поле на форме Составление платежных поручений),

- в субконто 3 заносится ставка подоходного налога - берется из записи таблицы Зарплата, на которую ссылается проводка.

 

Для перечисления с типом Единый социальный налог  данные собираются по виду докспец, указанному в соответствующей настройке ЗП Перечисления. Спецификации по сотрудникам создаются так:

- в субконто 1 заносится Сотрудник - берется из субконто 1  докспец, по которой собираются данные,

- в субконто 2 заносится Расчетный месяц (значение, которое указывается в соответствующем поле на форме Составление платежных поручений).

 

Примерный план настройки платежных поручений:

 

После обновления версии, синхронизации базы данных эталоном текущего релиза и импорта конфигурации BuhtaConfig следует:

1. Создать три новых вида спецификаций для вида документа Платежное поручение-.

Примерные названия этих докспец: Подоходный по сотруднику, ОПС страх по сотруднику, ОПС накоп по сотруднику.

Флаг Входит в сумму не выставлять.

2. В карточках редактирования ЗП Перечисления:

• для перечисления с типом Подоходный налог установить флаг Создавать докспец с суммами перечислений по каждому сотруднику и для параметра Вид докспец Перечисление по сотруднику выбрать из справочника вид докспец Подоходный по сотруднику.

•для перечисления с типом Единый социальный налог и видом проводок ОПC страх часть установить флаг Создавать докспец с суммами перечислений по каждому сотруднику, для параметра Вид докспец Перечисление по сотруднику выбрать из справочника вид докспец ОПС страх по сотруднику, а для параметра Суммы к перечислению искать по виду докспец выбрать из справочника значение вида докспец ОПС: Страховая часть.

•для перечисления с типом Единый социальный налог и видом проводок ОПC накоп часть установить флаг Создавать докспец с суммами перечислений по каждому сотруднику, для параметра Вид докспец Перечисление по сотруднику выбрать из справочника вид докспец ОПС накоп по сотруднику, а для параметра Суммы к перечислению искать по виду докспец выбрать из справочника значение вида докспец ОПС: Накопительная часть.

3. В основной настройке ЗП:

• на закладке Подоходный налог для параметра Виды докспец-перечислений налога по сотрудникам выбрать из отфильтрованного справочника значение вида докспец Подоходный по сотруднику,

•на закладке Страх.взносы для параметра на ОПС Страх.часть по сотрудникам выбрать из отфильтрованного справочника значение вида докспец ОПС страх по сотруднику

•на закладке Страх.взносы для параметра на ОПС Накоп.часть по сотрудникам выбрать из отфильтрованного справочника значение вида докспец ОПС накоп по сотруднику
