Безопасность, пароли, доступы

 

[ * ] [17855], [17888], [17896]

Внесены изменения в следующие права:

•Право Справочники - Юридические лица.

В предыдущих версиях право Юридические лица отвечало за возможность входа в программу под указанным юр.лицом. Начиная с текущего релиза, если у пользователя нет прав на какое-нибудь юр.лицо, он не только не может входить под этим юр.лицом в приложение, но и просматривать данные по этому юр.лицу в справочниках и оборотной ведомости работая под другим юр.лицом (ранее такая возможность была, если в настройке запроса или в настройке оборотной ведомости использовался флаг Отображать данные для указанных юр.лиц).

 

•Право Сервис - Настройка программы - Отображать данные для указанных юр.лиц.

Если у пользователя нет права Отображать данные для указанных юр.лиц, то:

- в карточках настройки запроса и оборотной ведомости соответствующая настройка на юр.лица становится недоступной (ранее при отсутствии этого права она была невидима);

- в справочниках и оборотной ведомости данные отображаются в соответствии с настройкой на юр.лица (ранее при отсутствии этого права данные отображались по текущему юр.лицу).

 

Документы

 

[ * ] [17809]

Установлено следующее правило для режима просмотра документов по "чужим" юр.лицам -

если в запросе, по которому строится список документов, установлена настройка Отображать данные для указанных юр.лиц, Список юр.лиц = Один,  то из списка документов возможен просмотр карточек редактирования (с возможностью просмотра спецификаций и проводок) по указанному юр.лицу, "чужому" в том числе, и копирование записей по этому юр.лицу в текущее юр.лицо.

Отчеты

 

[ + ] [17544]

•Изменены бухгалтерские отчеты Бухгалтерский баланс и Отчет о прибылях и убытках. На форму [Стандартная. Бухгалтерский баланс 2011] добавлен признак "в форме, рекомендованной ФНС РФ". Если он выставлен, то отчеты Бухгалтерский баланс и Отчет о прибылях и убытках составляются по новому шаблону [Стандартный. Бухгалтерская отчетность 2011], если не выставлен - по шаблону [Стандартный. Бухгалтерский баланс 2011].  Все объекты, необходимые для построения этих отчетов, включены в файл конфигурации BuhtaForm.mdb.

Доступна выгрузка отчетов в XML-файл, составленных по шаблону [Стандартный. Бухгалтерская отчетность 2011] - обновлены формы Стандартная. Архивный отчет в электронном виде] и [Стандартная. Архивный отчет в электронном виде_91], которые также входят в файл BuhtaForm.mdb.

 

•Изменен Отчет о движении денежных средств. Теперь он вызывается из формы [Стандартная. Отчет о движении денежных ср-в 2011] и строится по шаблону [Стандартный. Отчет о движении денежных ср-в 2011]. Настройка отчета за отчетный период производится по запросу [Стандартный. Форма Ф4_2011/1], за соответствующий период предыдущего года - по запросу [Стандартный. Форма Ф4_2011/2].

Все объекты, необходимые для построения Отчета о движении денежных средств, включены в файл Стандартной конфигурации BuhtaForm.mdb.

В поставку текущей версии входит также обновленный файл Пример_настройки_ФО.mdb, в который добавлены строки с примерами настроек для новой формы Отчета о движении денежных средств (код формы Ф4_2011).

Пользователю через конфигуратор следует произвести импорт файла BuhtaForm.mdb и перезагрузить приложение. Затем, воспользовавшись пользовательской формой Импорт настроек ФО, закачать данные для формы Ф4_2011, запустить форму [Стандартная. Отчет о движении денежных ср-в 2011] и приступить к настройке отчетов.

 

Существенные отличия в настройке отчета по сравнению с предыдущей версией заключаются в следующем:

- добавлены настройки запросов за предыдущий период; таким образом, перед составлением отчета необходимо настроить два запроса: за отчетный период, и за соответствующий период предыдущего года;

- добавлена возможность сохранения рассчитанных показателей в таблицу [_Показатели из архивных отчетов] и их дальнейшего использования для настройки запросов за предыдущие периоды.

 

Рекомендации по настройке запросов.

-  Запросы за предыдущие годы можно рассчитывать по сальдо оборотам, выбрав в карточке настройки для параметра Тип значения Дебет, Кредит, а можно брать из таблицы [_Показатели из архивных отчетов], в этом случае для параметра Тип указывается значение Показатели из отчетов. При этом в случае настройки Отчета о движении денежных средств по каждой строке выбираются значения [Ф4_2011 рзд. стр.<номер строки> грф.1] для формы с кодом Ф4_2011. Воспользоваться таблицей [_Показатели из архивных отчетов] можно в том случае, если она содержит нужные показатели за предыдущие годы. Для этого достаточно составить Отчет о движении денежных средств за 1 квартал 2010 года, 1 полугодие 2010 года, 9 мес 2010 года и 2010 год, и при его сохранении на вопрос о сохранении данных в таблицу показателей ответить Да.

-  Значение Формула параметра Тип позволяет использовать для настройки одних показателей значения других показателей. Так например, если в поле Тип установить значение Формула, для показателя 4100, который является суммой показателей 4110 и 4120, можно указать значение s4110+s4120.

-  Можно комбинировать типы настроек и фильтров в пределах одного показателя. Так если требуется для строки с кодом 4121 собрать данные по Дебету счетов 60, 71, 76 и по Кредиту счета 60, заводится два показателя: один - 4121а со значением Дебет в поле Тип и значениями 60,71,76 в поле Список счетов, второй - 4121b со значением Кредит в поле Тип и значением 60 в поле Список счетов.

 

Напоминаем, что после изменения настроек запроса или импорта показателей с помощью формы Импорт настроек ФО, следует при выходе из окна со списком настроек нажать на кнопку <Сохранить>. В противном случае список настроек не будет соответствовать запросу, по которому составляется отчет.
