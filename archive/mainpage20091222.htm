Настройка

 

[ * ] [16524]

Изменен параметр Использовать стандартный список для выбора вида при добавлении записи из общей настройки программы (меню Настройка - Общая настройка, закладка Общие настройки) - теперь выставление флага для этого параметра  может влиять не на все таблицы, имеющие специальную форму выбора вида, а только на выбранные из списка, расположенного здесь же в общей настройке.        

Если установлен флаг Использовать стандартный список для выбора вида при добавлении записи и среди выбранных таблиц имеются таблицы Докспец, Догспец, Проводка или указано значение Все, то при добавлении новой записи в эти таблицы в списках для выбора вида производится автоматическая фильтрация по виду Документа / Договора.

[ + ] [16540]

В пользовательских формах для компонента TbuhtaTreeList добавлена возможность настройки видимости колонок списка - пункт контекстного меню списка Настройка колонок. Настройки сохраняются для каждого рабочего места пользователя и не зависят от его логина.

Для того чтобы пункт меню Настройка колонок отобразился, необходимо проинициализировать параметр Настройка колонок:

 Param.Values['Настройка колонок']:= True;

где Param - свойство объекта TbmForm.

[ * ] [16067]

1. В соответствии с Федеральными законами N 212-ФЗ и N 213-ФЗ от 24.07.2009 внесены изменения в начислении страховых взносов по сравнению с начислением ЕСН:

1.1. Налог ПФР не начисляется

1.2. В 2010 г. тарифы взносов в ФСС, фонды медицинского страхования (ТФОМС и ФФОМС) и на обязательное пенсионное страхование (ОПС) не меняются

1.3. В 2010 г. остаются льготы для предприятий, облагающихся:

- по упрощенному налоговому режиму, по вмененному доходу – облагаются только ОПС, по пониженному тарифу

- для сельхозпроизводителей – облагаются по пониженному тарифу для с/х производителей

1.4. С 2010 г льгота для инвалидов действует на всю сумму дохода (не ограничивается 100000 руб.)

1.5. С 2010 года доходы иностранцев не облагаются страховыми взносами во все фонды

1.6. С 2011 года введены новые тарифы страховых взносов, отменяются льготы для предприятий, облагающихся по упрощенному налоговому режиму и по вмененному доходу

1.7. Для предприятий-сельхозпроизводителей, предприятий, действующих в спец.экономических зонах и для инвалидов введен порядок постепенного перехода на новые тарифы – в 2011-2012 и в 2013-2014 гг действуют пониженные тарифы.

1.8. С 2010 года отменена регрессивная шкала – теперь страховыми взносами облагается вся сумма дохода до 415 000 руб. На сумму дохода, превышающую размер 415 000 руб. за год, страховые взносы не начисляются. Максимальный размер облагаемого дохода может меняться.

2. В связи с этим изменен алгоритм расчета ЕСН (страховых взносов)

2.1. В таблицу [Ставки на дату] добавлена Ставка «Макс.база для страх.взносов»=415 000 c 01/01/2010 - ограничивает размер дохода, на который начисляются страховые взносы.

2.2. Теперь сумма взносов рассчитывается отдельно по каждому виду дохода ЕСН – одна сумма по общему режиму, и отдельно по инвалидам, вмененному доходу и по упрощенному режиму. Это сделано для корректного расчета взносов по общему и пониженному тарифу и правильного сбора данных для печати отчетов. Для каждого вида дохода ЕСН – инвалидов, вмененного дохода и упрощенного режима – в документе «Расчетный лист ЕСН» создается отдельная позиция спецификации – начисление взноса, в которой в первом субконто проставлен Сотрудник, а во втором – соотв. Вид дохода ЕСН. Таким образом, общая сумма взноса складывается из сумм четырех позиций спецификации расчетного листка ЕСН – суммы взноса по общему режиму и сумм взносов по доходу инвалида, по вмененному доходу и по доходу, облагаемому по упрощенному режиму. Начиная с 2011 г страховые взносы по вмененному доходу и по упрощенному режиму начисляются по общему тарифу, и соответствующие им позиции спецификации создаваться не будут. Для корректного разнесения сумм страховых взносов по затратам, для каждой позиции спецификации расчетного листа ЕСН с начислением взноса создаются отдельные позиции спецификации с разнесением суммы взноса по затратам. В поле «Процент» каждой позиции спецификации указывается процент, в соответствии с которым начислен взнос.

3. В таблице [Вид дохода ЕСН] добавлен вид дохода 13 «Доходы инвалидов».

3.1. Проставляется при расчете ЗП в шифрах расчетного листка ЗП, если в таблице ЗП Налоги сотрудника на начало расчетного месяца проставлен признак «Назначить/отменить льготу по ЕСН» «Льгота инвалидам по ЕСН и др.налогам с ФОТ». В шифрах с видами дохода ЕСН = 1-2-6-7-8-9 вид дохода ЕСН 13 «Доходы инвалидов» проставляется в поле «Вид дохода ЕСН», а в шифрах с видом дохода ЕСН = 3-4-5-10-11-12  в поле «Тариф ЕСН».

3.2. Проставляется при расчете ЕСН в шифрах расчетного листка ЗП,  в которых не проставлен признак «Вид дохода ЕСН» по тому же алгоритму, как в п. 3.1.

4. В таблице [Зарплата] и в таблице [Шифр ЗП] добавлено поле «Тариф ЕСН», которое является ссылкой на таблицу [Вид дохода ЕСН] и может принимать значения 7 «Вмененный доход», 8 «Упрощенный режим», 13 «Доходы инвалидов». Это поле используется для шифров с видом дохода ЕСН 3, 4, 5, 10, 11, 12 (доходы, которые облагаются взносами не полностью или не облагаются совсем, но должны попасть в отчеты о доходах и взносах) – для указания, по какому тарифу начислять страховые взносы в случае превышения суммы дохода над суммой вычета (для видов дохода ЕСН 3,4,5) и в какую графу отчета должна попасть сумма шифра (для видов дохода ЕСН 10,11,12).

Рекомендуется указывать «Тариф ЕСН» в тех шифрах ЗП с видом дохода ЕСН 3-4-5 или 10-11-12, которые облагаются страховыми взносами по пониженному тарифу – как доходы инвалида или вмененный доход или по упрощенному режиму.

Поле «Тариф ЕСН» используется также при разделении суммы шифра на две – облагаемую по вмененному доходу и по общему режиму (если в настройке карточки шифра проставлен соотв.признак). Тогда при расчете зарплаты шифр делится на два шифра, в одном проставляется вид дохода ЕСН из карточки шифра, а в другом – вид дохода ЕСН = 7 «Вмененный доход». Раньше возникали проблемы с шифрами с видами доходов ЕСН 3-4-5 – вид дохода ЕСН заменялся на 7 и расчет ЕСН по таким шифрам производился без учета вычета. Теперь при делении таких шифров на два вид дохода ЕСН в обоих шифрах проставляется в соотв. с видом дохода ЕСН в карточке, но в  поле «Тариф ЕСН» в одном из них проставляется 7 «Вмененный доход».
