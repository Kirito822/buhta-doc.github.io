Внесены изменения в процедуру формирования документа из договора по алгоритму Частичная спецификация и Частичная спецификация (кроме отгруженных). Поля Сумма и Сумма НДС округляются до двух знаков после запятой.

 

 

 

Справочники

 

[ + ][21914]

 

В справочник Организация добавлены поля для ввода адреса по справочнику ГНИ.

 

Для того, чтобы поля были доступны в карточке редактирования организации, необходимо:

 

1. меню Справочники->Настройка->Организации->Общая настрйока на закладке Главная проставить новый признак Использовать справочники ГНИ для заполнения адреса"

2. В настройке карточки организации открыть одно из новых полей (Адрес Регион, Адрес Район, Адрес Город, Адрес Улица)

3. Тогда в карточке организации появляется новая закладка Адрес по справочнику ИМНС, на которой по нажатию на кнопку Открыть справочники ИМНС можно заполнить элементы адреса в соответствии со справочниками ИМНС. По нажатию на кнопки со стрелками можно заполнить старые поля Адрес и Юр.Адрес строкой, сформированной по значениям этих новых полей.
