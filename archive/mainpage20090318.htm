Договоры и документы

 

[ * ] [15410]

Изменена карточка копирования документа (меню Данные - Документы, пункт Копировать контекстного меню списка документов):

•Добавлен флаг Выполнить генерацию. Если этот флаг установлен и выставлены флаги Копировать спецификации и Копировать зарплату, выполняется генерация новых проводок в соответствии с настройками генерации по имеющимся записям спецификаций и зарплаты.

•При копировании таблицы Проводка (на карточке копирования выставлен флаг Копировать проводки) теперь не копируется значение поля Зарплата. Чтобы проводки содержали ссылки на таблицу Зарплата, копирование документов следует проводить с выставленным флагом Выполнить генерацию.

 

Книга покупок и книга продаж

 

[ - ] [15265]

Исправлена ошибка в Дополнительных листах Книги покупок и Книги продаж.  Выборка строк книги для подсчета итогов в Дополнительном листе осуществлялась с даты начала года для заданного периода. Не попадали в итоги записи с исходной датой включения от предыдущего года. Теперь начальная дата выборки для итогов устанавливается по минимальной дате из следующих значений: даты начала года и минимальной исходной даты печатаемого списка.

 

Справочники

 

[ * ] [15308]

Внесены изменения в режим настройки иерархии справочников (меню Справочники - Настройка - Иерархия в справочниках):

•Добавлен новый параметр в таблицу настроек Поле сортировки видов. Это поле становится доступным для заполнения, если для таблицы установлена настройка иерархии по виду (выставлен флаг в столбце По виду). Значение, которое задается в столбце Поле сортировки видов,  определяет сортировку таблицы видов в левой части окна справочника. По умолчанию сортировка производится по полю Ключ.

• Закрыта возможность изменения иерархической структуры справочника - перемещения элемента из одного узла дерева иерархии в другой методом  перетаскивания (drag-and-drop).

 

[ - ] [15373], [15384]

В справочнике Счетов (меню Настройка - План счетов) исправлены ошибки вызова следующих режимов:

- Карточка движения по субконто (контекстное меню Сервис или "горячая" клавиша F7),

- Переслать в сообщении (контекстное меню Сервис),

Примечание: режимы становились доступны, если таблица Счет была включена в справочник Типы субконто (меню Настройка - Типы субконто).

 

•Поскольку Счет не может участвовать в качестве субконто, то вызов Карточки движения по субконто теперь заменен на вызов Карточки движения по счету. Этот режим доступен всегда, независимо от того входит ли таблица Счет в справочник Типы субконто или нет.

•Исправлен режим Переслать в сообщении (как и ранее он доступен только в том случае, если таблица Счет входит в справочник Типы субконто), при его вызове открывается окно с сообщением, в котором передается номер счета.

 

Расчеты

 

[ - ] [15309]

Внесены изменения в режим Расчета износа (меню Расчеты - Износ ОС и НМА). Новый расчет ежемесячной суммы выполняется теперь не только при изменении балансовой стоимости, как было ранее, но и при изменении поправочного коэффициента или срока использования (нормы износа).

При работе с личными карточками сотрудников в режиме Зарплата, а именно при сохранении записей ЗП Статуса с типом Увольнение, добавлены предупредительные сообщения о пересекающихся с датой увольнения ЗП Статусах и ЗП Инфо. Информация для разработчиков

 

[ + ] [15411]

В VB-скрипт к стандартной форме просмотра договоров добавлены события OnCreateDoc и OnAfterCreateDoc.

В форме на эти события могут быть созданы процедуры:

-  FormOnCreateDoc(ByRef DocVid, ByRef DocDate, ByRef PartialSpec, ByRef CreateAllow, ByRef CreateAllowMsg) - срабатывает для каждого отмеченного договора + для каждого вида документа;

- FormOnAfterCreateDoc(DocID)  - срабатывает для каждого  полученного документа.
