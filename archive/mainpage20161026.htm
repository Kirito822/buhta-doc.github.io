Договоры и Документы

 

[ * ] [21871]

 

В форму Локальные настройки на закладку Договоры\Документы

добавлен параметр Не снимать отметку строк при создании документов из списка договоров.

По умолчанию параметр установлен в значение False.

Если параметр выставлен в значение True, после окончания работы режима Создание документов список отмеченных записей не очищается.

 

 

Отчеты

 

[ + ] [21875]

 

При составлении отчета типа FastReport с помощью функций ShowReportWithParam доступен новый параметр TryToConvert. Параметр работает в сочетании с параметром ExportType, равным XLS.

Значение по умолчанию True.

Значение False указывает, что все данные будут выгружены в файл XLS в текстовом формате.

 

[ - ] [21865]

 

В режиме составления Авансового расчета по транспортному налогу в графу 9  вместо понижающего коэффициента ошибочно выводился повышающий коэффициент, добавленный по заявке 21493 в релизе 176.

 

[ * ] [21877]

 

В режиме составления Налоговой декларации налога на имущество из-за изменения кодов периодов по заявке 21734 перестали выводится данные по расчету стоимости имущества за 2 и 3 кварталы на 3 листе.

Информация для разработчиков

 

[ + ] [21875]

 

При составлении отчета типа FastReport с помощью функций ShowReportWithParam доступен новый параметр TryToConvert.

ShowReportWithParam

 

var

 fParam: TbuhtaParam;

 SQL: string;

begin

 fParam := NewParam;

 fParam.Value('RecordId') := 15;

 fParam.Value('WithoutPreview'):=True;

 fParam.Value('SaveTo'):='...\Пример.xls';

 fParam.Value('ExportType'):='XLS';

 fParam.Value('TryToConvert'):=false; //новый параметр

ShowReportWithParam('Стандартный. Список дебиторов и кредиторов', fParam, Self);

 fParam.Free;

end;
