Автонумерация

 

[ + ] [16567]

К списку объектов для экспорта/импорта через Конфигуратор (меню Сервис - Конфигуратор) добавлена Автонумерация.

В связи с этим изменена структура таблицы Автонумерация - добавлены стандартные поля, используемые при экспорте/импорте конфигурации:

- Конфигурация

- Версия

- Кто изменил

- Когда изменил

- Что изменил

- Исх. конфигурация

- Исх. версия

- Исх. ключ

- Группа.

Поля Конфигурация, Версия, Кто изменил и Когда изменил автоматически заполняются при создании новой автонумерации, поля Кто изменил и Когда изменил автоматически обновляются при изменении автонумерации.

 

В Конфигураторе отражается Автонумерация для объектов, не имеющих разделение по юр.лицам, а также Автонумерация по текущему юр.лицу для объектов, имеющих разделение по юр.лицам.  

 

ОС и НМА

 

[ + ] [16559]

В настройку расчета износа ОС и НМА (режим Справочники - Настройка - ОС и НМА - Настройки расчета износа) добавлена возможность указания пользовательской процедуры расчета износа (ранее расчет износа осуществлялся всегда стандартной процедурой Расчет_Износа).

Выбор используемой процедуры осуществляется на закладке Прочее в форме Настройка расчета износа, параметр Процедура расчета износа. Значение, устанавливаемое по умолчанию - стандартная процедура Расчет_Износа.

Требования к пользовательской процедуре:

1. Процедура должна входить в состав объектов SQL (см. меню Сервис - Объекты SQL).

2. Имя пользовательской процедуры должно содержать строки символов расчет и износ или расчет и аморт (например, [_УСН Расчет износа] или  [Расчет сумм амортизации ОС]).

3. Пользовательская процедура должна иметь тот же состав параметров, что и стандартная процедура Расчет_Износа.

Оборотная ведомость

 

[ * ] [16583]

В общую настройку (режим Настройка - Общая настройка) на закладку Оборотная ведомость добавлен флаг Отображать свернутые в ноль обороты.

Если флаг выставлен, то в оборотной ведомости отображаются строчки с нулевыми значениями в полях: Вх. сальдо, Дебет, Кредит, Исх. сальдо.

 

Настройка

 

[ * ] [16642]

В форму Общая настройка (меню Настройка - Общая настройка) на закладку Главная добавлен параметр Считать первую неделю года с вариантами: С первого дня года, С первого четверга.

Эта настройка используется стандартной функцией dbo.[Название недели], а также согласно этой настройке определяется номер недели для поля типа Период.

Если установлено значение С первого дня года, первой неделей года является неделя (в том числе неполная) с 1 января по первое воскресенье, последней неделей года - неделя (в том числе неполная) с последнего понедельника по 31 декабря.

Если установлено значение С первого четверга первой неделей года является полная неделя с понедельника по субботу, в которую включен первый четверг года.

Значение по умолчанию - С первого дня года.

[ * ] [16632]

В карточку настройки Вида статуса ЗП (меню Зарплата - Настройка - Виды статусов ЗП) добавлен флаг Не отображать параметры предыдущих статусов.

Для статусов, в карточке вида которых установлен флаг Не отображать параметры предыдущих статусов, при добавлении и редактировании статуса информация в карточке ЗП статуса о параметрах предыдущих статусов не выводится, все параметры заполнены значениями нет данных. Для других видов статусов поиск последних параметров производится как и раньше - по последней дате, но при этом ЗП статусы, в карточке вида которых установлен флаг Не отображать параметры предыдущих статусов, игнорируются.

Измененная форма [Стандартная. Вид статуса ЗП (карточка)] для настройки Вида статуса ЗП входит в конфигурацию Стандартная_Зарплата_203.

 

[ * ] [16646]

В соответствии с приказом  ФНС РФ от 22 декабря 2009 г. N ММ-7-3/708@  в справочники Комплекса БУХта добавлены новые коды доходов и вычетов:

•коды дохода: 1535, 2791.

•коды вычетов: 318, 319, 509, 607

 

Для того чтобы новые виды вычетов учитывались при расчете подоходного налога, следует: после синхронизации базы данных новым эталоном добавить их в таблицу Ставка по дате (меню Зарплата-Настройка-Ставки на дату), добавить ставку с типом Налоговый вычет, в поле Код указать код вычета (318,319 и т.д.), в поле Значение указать сумму вычета, в поле Дата указать дату начала действия вычета.
