Книга покупок и книга продаж

 

[ + ] [20885]

В Документы / Договора добавлена возможность указания данных о Посреднике и отражение этой информации в Книгах покупок / продаж.

В рамках этой модернизации внесены следующие изменения:

•В таблицы Документ и Договор добавлены поля:

- Тип посредника (возможные варианты значений Нет,Орг,Чел)

- Посредник.

Указанные поля в карточках редактирования документов, договоров отображаются по настройке полей.

•В Книгах покупок/продаж поле Посредник заполняется значением соответствующего поля из Документа (Договора) в следующих режимах:

- Данные – Книга покупок, Книга продаж, кнопка <Составить книгу>.

- Контекстное меню списка документов, пункты Сервис – Добавить строку в книгу продаж, Добавить строку в книгу покупок

- Карточка редактирования документа, кнопка <Доп.режимы>, кнопки <Добавить запись в книгу продаж>, <Добавить запись в книгу покупок>.

Модернизация требует синхронизации БД  эталоном текущей версии.

Отчеты

 

[ * ] [20728]

Обновлены следующие формы статистического наблюдения:

•Основные сведения о деятельности организации, форма № 1-предприятие:

Начиная с отчета за 2014 используется новый шаблон [Стандартный. Осн.сведения о деят-сти 1-предпр 2014], в котором задействованы дополнительные параметры. Отчетная форма также изменила свой внутренний настроечный код, теперь он называется не 1предпр, а 1пр2014. Поэтому помимо  импорта файла конфигурации BuhtaForm, в который входят новые и измененные в рамках текущей модернизации объекты, в поставку этой версии включен обновленный файл Пример_настройки_ФО.mdb, в который добавлены показатели для нового запроса [Стандартный. Форма 1пр2014/1]. Пользователю после импорта конфигурации BuhtaForm и перед составлением отчета следует импортировать настройки форм отчетности с кодом новой формы 1пр2014 из файла Пример_настройки_ФО.mdb с помощью пользовательской формы [Импорт настроек ФО], а затем через обновленную пользовательскую форму [Стандартный. Осн.сведения о деятельности 1-предпр] настроить запрос  для формы отчетности 1пр2014.

•Сведения об основных показателях деятельности малого предприятия, форма № ПМ:

Исправлена  форма [Стандартный. Сведения о деят-сти малого предпр. ПМ] - в Разделе 3 отчета убрана таблица, оставлен только подвал "Справочно".

•Сведения о производстве и отгрузке товаров и услуг, форма № П-1:

Исправлены  форма [Стандартный. Сведения о производст. и отгрузке П-1] и шаблон [Стандартный. Сведения о производст. и отгрузке П-1] – внесены изменения в Раздел 1 отчета, строки 11,12.

Добавленные  и измененнные объекты входят в состав конфигурации BuhtaForm.

 

[ * ] [20871]

Добавлен вызов режима Обработчик (выбор обработчика из списка HotKey пользовательских форм) для:

- Оборотной ведомости,

- Сводного отчета (при выставленном параметре настройки Преобразов. для вызова карточки),

- Карточки движения по субконто,

- Карточки движения по счету.

Вызов окна обработчика осуществляется из контекстного меню указанных режимов, по нажатию на кнопку Wisard Главной инструментальной панели или с помощью "горячих" клавиш Alt+F12. Также в указанных режимах срабатывают "горячие" клавиши самих HotKey форм.
