Договоры и документы

 

[ - ] [18815]

При копировании документа в документ (договор) или договора в договор (документ) с частичным копированием спецификации не копировалось время у пользовательского поля таблицы Докспец (Догспец) с типом ДатаВремя. Ошибка исправлена.

 

Книга покупок и книга продаж

 

[ - ] [18897]

В карточки редактирования разделов Книги покупок, Книги продаж (меню Данные - Настройка - Настройка книги покупок, Данные - Настройка - Настройка книги продаж) и Книги доходов, расходов (меню Данные - Настройка - Настройка книги учета доходов и расходов) возвращены кнопки <Сохранить>, <Отмена>, которые были ошибочно удалены в релизе 132.

 

[ * ] [18923]

В журнале учета и в карточке редактирования выставленных счетов-фактур (меню Данные - Книга продаж, кнопка <Журнал>) переименованы соответственно столбец и поле, содержащие данные о наименовании покупателя - значение Поставщик изменено на Покупатель. Обновлены запрос [Журнал учета выставленных с/ф] и форма [Книга продаж (Карточка Журнала учета с-ф)] входящие в стандартную конфигурацию BuhtaConfig.

 

[ * ] [18885]

Доработан алгоритм заполнения Журнала выставленных счетов-фактур (меню Данные - Книга продаж, кнопка <Журнал>) - в том случае, когда счета-фактуры выставляются на организацию, совпадающую с организацией продавца, в журнале в поле Покупатель проставляется наименование этой организации (раньше указывалось значение <Нет>).

 

Курсы валют

 

[ - ] [18854]

Исправлены пользовательские формы [Импорт курсов валют],  [Курсы валют], которые входят в состав стандартной конфигурации BuhtaConfig:

- изменен формат передаваемых данных - символы кириллицы в разделителях заменены на латинские;

- добавлено сообщение о невозможности импорта, в случае если в справочнике валют не указано ни одной записи с признаком Загружать курс через Интернет;

- откорректирован режим оповещения сотрудников о загрузке / изменении курса валют.

 

Кадры

 

[ - ] [18926]

Исправлена ошибка при заполнении Табеля по ЗП Инфо. В том случае когда в соответствующей записи ЗП Инфо не указано количество отработанных часов, при неполном рабочем дне данные для табеля должны браться из Статуса сотрудника. Алгоритм не срабатывал, если Дата начала в ЗП Инфо приходилась на 1-ое число месяца.

 

Безопасность, пароли, доступы

 

[ * ] [18915]

На форму Пользователи / Пароли / Доступы в список Учетных записей добавлена колонка Сотрудник.

 

[ * ] [18928]

На встроенную форму Пользовательские доступы (меню Настройка - Пользователи / Пароли / Доступы, кнопка <Пол. доступы>) добавлены кнопки <Добавить>, <Изменить>, <Удалить>.

 

Информация для разработчиков

 

[ + ] [18890]

В палитру компонент форм пользователя на закладку БУХта добавлен новый компонент WebBrowser для возможности вызова веб-обозревателя из пользовательских форм комплекса БУХта.

 

Пример скрипта с вызовом веб-сайта www.yandex.ru

 

procedure TSimpleForm.bmFormShow(Sender: TObject);

begin

WebBrowser1.Navigate('www.yandex.ru');

end;
