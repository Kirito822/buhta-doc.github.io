Отчеты

 

[ * ] [15800]

Доработан режим отправки по почте отчета, созданного на основе шаблона типа FastReport версии 4

(режим вызывается из окна предварительного просмотра отчета через инструментальную панель или контекстное меню отчета - кнопки инструментальной панели и пункты меню Отослать по E-mail или Экспорт -> E-mail)).

Режим модернизирован для экспорта с использованием почтового клиента (в окне Отослать по E-mail выставлен флаг Использовать почтового клиента). Ранее при этом варианте экспорта настройки электронного адреса (Кому, Копия, Скрытая копия), темы сообщения и текста сообщения заполнялись в окне просмотра сообщения. Теперь добавлена возможность их заведения в окне экспорта Отослать по E-mail.

 

[ * ] [15921]

Расширена настройка показателей в Отчете о движении денежных средств (вызывается через пользовательскую форму [Стандартная. Отчет о движении денежных средств_91]) - добавлен фильтр по корр. счетам.

Измененные в рамках заявки объекты входят в конфигурацию BuhtaForm.mdb.  Изменен также файл Пример_настройки_ФО.mdb  (находится в каталоге Дополнения) с примерами настроек отчета.

 

Зарплата См. также Конфигурация ''Стандартная_Зарплата_...''

Настройка

 

[ * ] [15852]

В меню Настройка - Локальные настройки на закладку Общие добавлен флаг В сообщениях не указывать адресата "Всем" по умолчанию. Если этот флаг выставлен, то при создании сообщения поле адресата остается незаполненным. При попытке отправить такое сообщение выдается предупреждение о необходимости указания имени пользователя-адресата. Если флаг не выставлен, то при создании сообщения поле адресата автоматически заполняется значением Всем.

 

[ + ] [15974]

В меню Настройка - Общая настройка на закладку Общие настройки добавлен новый параметр - текстовое поле В режиме "Файл" не применять архивацию для файлов следующих расширений (указать через запятую).

Для файлов с расширениями, которые пользователь укажет в этом поле, не будет действовать режим архивации, который применяется автоматически в стандартном режиме привязки файлов (Alt+F). Рекомендуется указывать расширения для файлов изображений (jpeg ...) или файлов являющихся архивом (rar, zip ...).

Файлы с расширениями doc, xls, rtf, csv подлежат обязательной архивации. Параметр настройки эти значения не сохраняет.

 

Синхронизация

 

[ * ] [15939]

Откорректирован режим импорта конфигурации при синхронизации баз данных. Добавлена дополнительная проверка на присутствие различий со статусом Конфликт или Пользователь между объектами, содержащимися в базе данных и в импортируемой конфигурации. Если такие различия имеются, то перед импортом пользователю задается вопрос В базе данных, среди стандартных объектов конфигурации, имеются объекты измененные пользователем. Выполнить импорт конфигурации? В случае отказа импорт не производится.

 

Информация для разработчиков

 

[ + ] [15978]

В VB-скрипт к карточке редактирования документа в событие FormOnDopRegimButtonClick добавлен флаг Allow.

По умолчанию Allow = True. В этом случае вызывается стандартная форма дополнительных режимов редактирования документа. Если Allow = False, то форма не вызывается.
