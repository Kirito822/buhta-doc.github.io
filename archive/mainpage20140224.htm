Документы

 

[ * ] [20106]

В соответствии с ПРИКАЗом МинФина РФ от 12 ноября 2013 г. N 107н об изменении реквизитов платежного поручения обновлены списки возможных значений полей Статус плательщика, Тип платежа, Основание платежа для следующих форм:

- Карточка редактирования ЗП Перечисление (вызывается из режима Зарплата - Настройка - Перечисление).

- Закладка Для налоговых платежей на карточке редактирования документа типа Стандартная или Платежное поручение (-). Для документов с датой >= 01.01.2014 списки для указанных полей помимо значений в соответствии с приказом содержат значения, введенные в эти поля в документах соответствующего вида с датой  >= 01.01.2014.

 

Книга покупок и книга продаж

 

[ * ] [19983], [20054]

Добавлены новые параметры для режимов настройки Книги покупок и Книги продаж:

 

•Контроль соответствия НДС и облагаемой суммы - если признак проставлен, то алгоритм контроля ошибок соответствует алгоритму, внесенному в 148 релизе в рамках заявки 19808. См. Релиз 148 Книга покупок и продаж. Если признак не выставлен проверка производится только для значения [Всего покупок с НДС] минус суммы без НДС минус суммы НДС.

•Полное название организации в "Наименовании покупателя" (для книги продаж) / в "Наименовании продавца" (для книги покупок) - если признак проставлен, то на экран и в отчет выводится полное название организации.

 

Модернизация требует синхронизации базы данных эталоном текущей версии для обновления процедур [Книга продаж показать результат] и [Книга покупок показать результат].

 

Справочники

 

[ - ] [20044]

В режиме удаления записи из таблицы Организация отсутствовала проверка на наличие ссылок на удаляемую запись в таблицах ТМЦ и Юр.лицо. Ошибка исправлена. Изменения внесены в триггер Организация_Удаление, что требует синхронизации базы данных эталоном текущей версии.

 

Отчеты

 

[ * ] [19936]

Изменен отчет Налоговая декларация по налогу на прибыль организаций.

- Начиная с 2013 года декларация теперь печатается по новому шаблону [Стандартный. Нал. декл.по налогу на прибыль 2013], в котором изменены штрих-коды и вместо аббревиатуры ОКАТО печатается ОКТМО.

- Добавлено заполнение Приложения № 5 по обособленным подразделениям.

- Добавлена выгрузка отчета, составленного по шаблону [Стандартный. Нал. декл.по налогу на прибыль 2013] в файл формата XML.

Все измененные и добавленные в рамках этой модернизации объекты входят в конфигурацию BuhtaForm.

Сервис

 

[ + ] [20029]

Модернизирован механизм отправки сообщений - добавлена возможность отправки сообщений с выбором списка адресатов, как активных, так и не работающих в данный момент в базе.

В стандартное окно отправки сообщений, наряду с уже имеющимся параметром Во все базы, добавлены два дополнительных: Всем в Бухте и Указать.

В следующей таблице приведены доступные возможности для выбора адресатов при различном сочетании указанных параметров во всех режимах системы Бухта, обеспечивающих отправку сообщений.
