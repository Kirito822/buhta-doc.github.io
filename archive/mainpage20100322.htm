Договоры и документы

 

[ - ] [16577]

В списках документов и договоров, когда в редакторе запроса выставлен флаг Отображать данные для указанных Юр.лиц, отображаются данные по другим юр.лицам. В этом случае для записей, относящихся к другим юр.лицам, были доступны следующие режимы:

 

В списке договоров:

- создание документов (кнопка на форме),

- бизнес-операции (кнопка на форме),

- корректировка дог.спец (кнопка на форме),

- изменение параметров договоров (пункт контекстного меню),

- установить/ снять блокировку (пункт контекстного меню),

- разделение договоров на два (пункт контекстного меню),

- доступ к пользовательским формам (пункт контекстного меню).

 

В списке документов:

- бизнес-операции (кнопка на форме),

- изменение параметров документов (пункт контекстного меню),

- установить/ снять блокировку (пункт контекстного меню),

- доступ к пользовательским формам (пункт контекстного меню).

 

Ошибка исправлена. Для записей, относящихся к другим юр.лицам, закрыт доступ к указанным режимам.

 

[ * ] [16681]

Модернизирован режим копирования документов со сменой юр.лица. Если в копируемом документе типы поставщика, получателя - Документ (Док) или Договор (Дог), то в создаваемом  документе соответствующие им  поставщик, получатель (т.е. документ или договор - относящиеся к юр.лицу, в данные которого производится копирование) ищутся по совпадению полей: [Номер], [Вид], [Дата], [Юр.лицо], [Тип поставщика], [Поставщик], [Тип получателя], [Получатель].

Если документ/договор не найден, поиск производится по полям: [Номер], [Вид], [Дата], [Юр.лицо].

Отчеты

 

[ * ] [16647]

Объединены режимы выгрузки архивных отчетов, которые производились из пользовательских форм [Стандартная. Архивный отчет в электронном виде] и [Стандартная. Архивный отчет в электронном виде_91] (входят в конфигурацию BuhtaForm).

Теперь все отчеты выгружаются из формы [Стандартная. Архивный отчет в электронном виде]  по клавише ALT+E. Клавиша ALT+R, по которой выгружались отчеты из формы [Стандартная. Архивный отчет в электронном виде_91] не работает, форма [Стандартная. Архивный отчет в электронном виде_91] теперь содержит только процедуры, которые вызываются из формы [Стандартная. Архивный отчет в электронном виде].

 

[ + ] [16682]

В форму запроса параметров для отчетов по Кассовой книге (меню Отчеты - Денежные средства - Кассовая книга) добавлена настройка Виды проводок для суммы "В том числе на зарплату". По указанным видам проводок производится заполнение суммы в строке отчета В том числе на зарплату .

 

[ * ] [16684]

Модифицированы отчеты из меню Отчеты - Расчеты с дебиторами и кредиторами:

•Дебиторская и кредиторская задолженность

- к кнопкам выбора счетов добавлены флаги Включая субсчета,

- изменен алгоритм расчета просроченных задолженностей, в связи с чем убрано поле Задолженность отсчитывается от,

- поле Количество дней отсрочки заменено на поле Количество дней отсрочки оплаты для дебиторов,

- добавлено поле Количество дней отсрочки оплаты кредиторам,

- счета долговременной задолженности теперь указывать не обязательно.

•Расшифровка дебиторской и кредиторской задолженности

- к кнопке выбора счетов добавлен флаг Включая субсчета,

- изменен алгоритм расчета просроченной задолженности, в связи с чем убрано поле Задолженность отсчитывается от,

- поле Количество дней отсрочки заменено на поле Количество дней отсрочки оплаты для дебиторов,

- добавлено поле Количество дней отсрочки оплаты кредиторам,

- добавлен флаг Без деления организаций по организационно-правовым формам.

•Список дебиторов и кредиторов

- к кнопке выбора счетов добавлен флаг Включая субсчета,

- поле Период заменено на поле Дата(начало дня) (раньше датой, на которую определялась задолженность, бралась дата конца периода),

- результат всегда сворачивается в целом по организации, независимо от настроек счетов в плане счетов,

- из шапки отчета убрана строка Существуют задолженности по счетам.

•Акт сверки взаиморасчетов

- к кнопке выбора счетов добавлен флаг Включая субсчета.

[ + ] [16644]

В соответствии с Приказом Минздравсоцразвития России от 12.11.2009 N 894н утверждена новая форма расчета РСВ-1 ПФР. В связи с этим  внесены следующие изменения:

•Для формирования отчета добавлена пользовательская форма [З/П Расчет по взносам на ОПС и ОМС РСВ-1 ПФР] (форма и связанные с ней объекты входят в состав конфигурации Стандартная_Зарплата_204).

•В карточку настройки юр.лица (меню Справочники - Настройка - Организации - Юридические лица) на закладку Отчетность в фонды добавлен параметр Регистрационный номер в ТФОМС, значение которого используется при составлении указанного выше отчета.

 

[ * ] [16668]

Внесены следующие изменения в карточки редактирования таблиц комплекса Зарплата:

•В  карточке редактирования настройки шифров-начислений и шифров-удержаний  (меню Зарплата - Настройка - Шифры ЗП) поле Процент на закладке Начисление/Алгоритм (Удержание/Алгоритм) становится видимым, если заполнено поле Алгоритм (убрана проверка на присутствие в названии алгоритма подстроки "процент")

•В карточке редактирования ЗП Инфо (меню Зарплата - Личные карточки сотрудников, закладка Опер.инф.) для шифров с типом Прочий открыто поле Календарь (если проставлен соответствующий признак в карточке настройки шифра).

 

[ * ] [16680]

В связи с заменой Единого Социального Налога (ЕСН) на Страховые Взносы (С/В) в режимы меню Зарплата внесены следующие изменения:

•Из меню Зарплата убран пункт Шкала ЕСН.

•Переименован режим Виды дохода ЕСН в Виды дохода С/В (ЕСН).

•Изменены названия видов дохода ЕСН и примечания.

•В окно Основной настройки З/П  (меню Зарплата - Настройка - Основная настройка) добавлена закладка Страх.взносы с параметрами для указания:

- видов докспец для начисления взносов (совпадают с докспец ЕСН на закладке ЕСН),

- признаков взятия взносов (совпадают с докспец ЕСН на закладке ЕСН),

- проводок по страховым взносам (перечисление ОПС, ФФОМС, ТФОМС, ФСС, возмещение ФСС и расходы на госстрах) для заполнения отчетов по СВ, эти поля отличаются от соответствующих полей для ЕСН на закладке ЕСН.

•В Списке Личных карточек сотрудника и в Личной карточке сотрудника если на экране выбран период начиная с января 2010 года и позднее, то на кнопках и в сообщениях вместо названия ЕСН выводится С/В.
