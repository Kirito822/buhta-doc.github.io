Автонумерация

 

[ * ] [18232]

В алгоритмы автонумерации добавлена дополнительная блокировка записей для устранения возможности дублирования номеров при пакетном создании объектов одного или нескольких видов в случае одновременной работы нескольких пользователей .

 

Расчеты

 

[ * ] [18143]

Модифицирован режим Закрытие (открытие) периода (меню Расчеты - Закрытие (открытие) периода).

- При запуске режима выдается сообщение всем пользователям о запрете работы с документами, при завершении режима - о разрешении работы с документами.

- При запуске режима Закрытие (открытие) периода в Журнал событий (меню Сервис - Журнал событий) автоматически добавляется запись об изменении оперативной даты. Для данной модернизации требуется обновление базы данных эталоном текущей версии,  в который входит измененная процедура [Новая оперативная дата].

 

Отчеты

 

[ + ] [18168]

Разработан шаблон [Стандартный. Карточка учета материалов М-17] для составления отчета Карточка учета материалов (форма М-17) - вызывается из списка ТМЦ. Шаблон и связанные с ним объекты, необходимые для составления отчета, входят в состав конфигурации BuhtaUniform.

 

Информация для разработчиков

 

[ + ] [18184]

Для объектов buhtaGrid и buhtaTreeList разработана возможность отображения  всплывающей подсказки по колонке. На событие OnMouseMove этих объектов следует повесить вызов следующих процедур:

 

procedure TLookForm.GridMouseMove(Sender: TObject; Shift: TShiftState; X, Y: Integer);

begin

 Grid.bmHitTest.Fill(X,Y);

 if Grid.bmHitTest.hitType = htColumn then

 begin

   Grid.Hint := Grid.Columns[Grid.bmHitTest.Column].Caption;

   Grid.ShowHint := True;

 end

 else

 begin

   Grid.ShowHint := False;

 end;

end;

 

procedure TLookForm.buhtaTreeList1MouseMove(Sender: TObject; Shift: TShiftState; X, Y: Integer);

begin

 buhtaTreeList1.bmHitTest.Fill(X,Y);

 if buhtaTreeList1.bmHitTest.hitType = htColumn then

 begin

   buhtaTreeList1.Hint := buhtaTreeList1.Columns[buhtaTreeList1.bmHitTest.Column].Caption;

   buhtaTreeList1.ShowHint := True;

 end

 else

 begin

   buhtaTreeList1.ShowHint := False;

 end;

end;
