Некоторые события имеют возвращаемый параметр Allow типа Boolean. В случае его значения = False дальнейшая обработка события не производится, действие вызвавшее это событие не осуществляется, в противном случае работа продолжается в штатном режиме.

 

Кроме вышеописанных событий,  в скрипте доступны события для всех компонентов этих форм типа TbmDBEdit:

         OnEnter

         OnExit

         OnChange

         OnValidate(ByRef IsValid, ByRef NoValidMessage)

На форме выбора цены появился флаг «Скрыть цены кроме выбранной», с помощью которого можно скрывать все колонки цен кроме выбранной. Значением этого флага можно управлять через скрипт.
