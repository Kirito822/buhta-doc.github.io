Инструкция по настройке и использованию режима «Импорт online из Бухты».

 

В релизе 67 был  изменен принцип работы BuhtaServer. В связи с этим использование режима «Импорт online из Бухты» требует дополнительных настроек, выполнить которые может только пользователь с правами администратора SQL-сервера.

1. Запустить на сервере Client Network Utility, назначить Alias для сервера.

2. Войти в настройку BuhtaServer из BuhtaClient.exe

3. Добавить в список баз те базы, между которыми предполагается устанавливать соединение online. При этом указывать в качестве сервера новый Alias, имена, соответственно, давать отличающимися от существующих.

4. Для использования режима импорта online необходимо:

 - входить в базу-получатель по имени нового подключения (через Alias)

 - при входе в режим выбирать из списка баз-источников только базы, подключенные через Alias.
