Книга покупок и книга продаж

 

[ - ] [20931]

При создании новой записи в ручном режиме (при нажатии на кнопку <Добавить>) в таблицах Книга покупок, Книга продаж, Журнал учета выставленных с/ф, Журнал учета полученных с/ф не проставлялось значение по умолчанию Нет в поле Тип посредника, в таблице Книга покупок - в поле Тип продавца, в таблице Книга продаж - в поле Тип покупателя. Ошибка исправлена.

 

Отчеты

 

[ - ] [20927]

При выгрузке в Excel отчетов, составленных по шаблону первого типа, или при сохранении этих отчетов в архив, данные, представленные в символьном формате, преобразовывались в числовой формат. Ошибка исправлена.

 

[ * ] [20937]

Внесены следующие доработки в режим экспорта в Excel:

1. Если в режиме Сводный отчет количество экспортируемых колонок превышает 255 (максимальное число колонок для Excel с форматом файла xls), выдается предупреждающее сообщение о невозможности экспорта (ранее производился аварийный выход из программы).

2. Для отчетов Карточка движения по субконто и Карточка движения по счету в контекстное меню добавлен пункт Экспорт в Excel.
