Автонумерация

 

[ * ] [20958]

Внесены изменения в процедуры автонумерации:

- Если в настройке автонумерации заполнен параметр Учитывать значение поля, то новый номер для записи ищется с учетом этой настройки только в том случае, если в процедуру автонумерации передано значение этого поля.

- Если значение не передано, то номер ищется так, как если бы параметр Учитывать значение поля был не заполнен.

 

Безопасность, пароли, доступы

 

[ + ] [20883]

Внесены дополнительные возможности в режим настройки прав доступа (режим Настройка - Пользователи / Пароли / Доступы, кнопка <Права>).

- Справочники - Аналитические справочники по видам - Подразделения / Виды подразделений

- Справочники - Аналитические справочники по видам - Приказы / Виды приказов

установлена выборочная настройка прав, то через контекстное меню таблицы, расположенной в правой части рабочего окна, доступны новые режимы обработки записей.

 

Если нет отмеченных записей, то новые режимы действуют для всех записей таблицы и меняют значение того столбца, на котором установлен курсор. Названия режимов: Всем - <Название столбца> - <Значение 1..N>.

Так если курсор установлен на столбце Только чтение, доступны следующие пункты контекстного меню:

•Всем - Только чтение  - Все
•Всем - Только чтение - Только свои
•Всем - Только чтение - Нет.

при выборе которых, в столбце Только чтение для всех записей устанавливается соответственно значения Все, Только свои и Нет, в зависимости от выбранного пункта меню.

 

А если курсор установлен на столбце Добавление - доступны пункты контекстного меню:

•Всем - Добавление - Все
•Всем - Добавление - Нет

и т.д. для всех остальных столбцов.

 

Если же в таблице настройки прав есть отмеченные записи, то в контекстном меню появляются пункты: Всем отмеченным - <Название столбца> - <Значение 1..N>. Режимы, которые будут вызываться при выборе этих пунктов, будут менять значения текущего столбца только у отмеченных записей.
