Договоры и документы

 

[ * ] [17822], [17889]

Модернизированы формы копирования документов и договоров.

- В режимах копирования документов и договоров  добавлена возможность изменения значения поставщика / получателя. Предоставляется выбор конкретного значения из справочника для указания новых значений или возможность замены значения поставщика на значение получателя и наоборот. При копировании со сменой юр.лица изменение поставщика / получателя  невозможно.

- В режим копирования документа добавлена возможность копирования частичной спецификации (по аналогии с режимом копирования договоров).

- На формы копирования документов, договоров добавлена кнопка <Запомнить настройки>. При ее нажатии в реестр сохраняются последние значения параметров на форме копирования. При следующих вызовах формы, до нового нажатия кнопки <Запомнить настройки> они будут устанавливаться по умолчанию.

 

[ * ] [17861]

Модернизирован режим создания документов из договоров.

 

В настройку видов договоров (меню Данные - Настройка - Виды договоров) на закладку Документы добавлена новая колонка параметров Дата по умолчанию, где по каждому виду документа указывается источник дат, которые задаются по умолчанию в режиме создания документов из договоров. Возможные варианты источников дат:

- Дата договора

- Текущая дата - системная дата

- Дата из признака - значение, установленное в поле Дата у признака, соответствующего этому виду документа.

 

В связи с расширением настроек вида договора для режима создания документов, изменен интерфейс формы задания параметров в этом режиме:

 

a) Пакетный режим создания документов из списка договоров по нажатию кнопки <Cоздание документов>.

- Расширен список вариантов для выбора источника дат. К уже имевшимся значениям Взять дату из признака договора и Значение добавлены варианты Взять дату договора и Взять дату из настройки вида договора (устанавливается в соответствии с указанными выше настройками вида договора по виду документа). Значение Взять дату из настройки вида договора устанавливается по умолчанию при открытии формы создания документов.

- Добавлена колонка Источник даты. Значения по каждому виду документа устанавливаются автоматически при смене значения в поле со списком режимов выборки дат. Помимо этого пользователь может сам установить для отдельных видов документов необходимый ему источник даты: дату договора, текущую (системную) дату, дату из признака, а также дату из поля ввода.

Замечание 1. Значение по умолчанию в поле ввода  - текущая (системная) дата.

Замечание 2. Если в поле Источник даты указано значение Дата из признака, а в настройках вида договора для этого вида документа нет настройки на признак документа, или поле Дата у этого признака не заполнено, то документ создается датой договора.

 

b) Режим создания документов из карточки редактирования договора на закладке Документы в режиме добавления новых записей.

Отличие формы задания параметров в этом режиме от аналогичной формы в пакетном режиме состоит в наличии столбца с указанием даты, который заполняется автоматически при смене значения:

- в поле со списком режимов выборки дат,

- в полях колонки Источник даты,

- в поле ввода даты, если в колонке Источник даты установлено значение Дата из поля ввода,

а также значение в столбце Дата может быть изменено непосредственным редактированием.    

Замечание 3. В отличие от пакетного режима в этом режиме датой по умолчанию в поле ввода даты является дата договора.

 

Описанная модернизация требует изменения структуры базы данных, поэтому при смене версии необходима синхронизация БД новым эталоном.

 

Отчеты

 

[ + ] [17829]

Добавлен новый отчет - унифицированная форма ИНВ-11 утвержденная постановлением Госкомстата РФ от 18 августа 1998 г. N 88.

Применяется при инвентаризации расходов будущих периодов.

Вызывается из пользовательской формы [Стандартная. Акт инвентаризации ИНВ-11], которая как и все объекты, необходимые для составления отчета (группа ИНВ-11), включена в состав конфигурации BuhtaUniform.mdb. Подробное описание см. раздел Конфигурация ''BuhtaUniform''.

 

[ * ] [17926]

Изменен состав пользовательских форм, которые вызываются из меню Отчеты - Формы бухгалтерской отчетности:

- При выборе пункта Бухгалтерский баланс или пункта Отчет о прибылях и убытках  вызывается одна и та же форма [Стандартная. Бухгалтерский баланс 2011], так как оба эти отчета теперь формируются в одном xls файле.

- При выборе пункта меню Отчет о движении ден.средств вызывается форма [Стандартная. Отчет о движении денежных ср-в 2011].

Безопасность, пароли, доступы

 

[ - ] [17626]

В настройках прав доступов для видов договоров и документов (режим Настройка - Пользователи / Пароли / Доступы, разделы Данные - Договоры / Виды договоров и Данные - Документы / Виды документов)  некорректно обрабатывалась настройка Чтение для Печатных форм - были доступны режимы Дизайнер и Настройка параметров страниц, а также возможность редактирования отчета. Исправлено для типов шаблонов 0 (FastReport 2), 1 (Эл.таблица) и 4 (FastReport 4).
