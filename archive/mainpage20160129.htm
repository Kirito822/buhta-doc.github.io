Настройка

 

[ + ] [21196]

Добавлена возможность автоматического отключения клиентов по времени простоя.

В карточку редактирования учетной записи пользователя (меню Настройка – Пользователи / Пароли / Доступы) добавлен новый параметр Максимальное время простоя программы, который указывает допустимое время простоя в мин. для текущего пользователя. Возможные значения от 10 мин до 2880 мин. Значение 0 (значение по умолчанию) означает, что время простоя не контролируется.

Работой пользователя считается обращение к SQL-серверу, нажатие клавиши и движение мыши при активном приложении.

По истечении указанного в настройке времени простоя выдается предупреждающее сообщение о предстоящем прерывании работы приложения с предложением дальнейшего продолжения работы.  При ответе <Нет> или отсутствии реакции со стороны пользователя приложение через несколько секунд закрывается, лицензия освобождается.  

В том случае, если пользователь по истечении времени простоя находится в открытых дизайнерах форм, шаблонов, запросов, фильтров, скриптов, в окне схемы данных, в окне настройки генерации проводок, работа не прерывается.

Изменение требует синхронизации БД эталоном текущей версии вследствие добавления в таблицу UserName поля IdleLimit.

В режиме работы с терминалами сбора данных (ТСД) возможность автоматического отключения по времени простоя не задействована.

[ * ] [21285], [21412] См. также раздел Стандартная_Зарплата

Внесены следующие изменения в структуру БД в соответствии с приказом ФНС России от 30.10.2015 № ММВ-7-11/485@.

•Обновлен Классификатор – добавлен новый вид Классификатора Статус налогоплательщика НДФЛ и в таблицу [Классификатор] записи 1-2-3-4-5-6 указанного вида  (исправлена процедура [Импорт справочника Кодов налогоплательщика ЕСН и ОПС]):

«1» - Налоговый РЕЗИДЕНТ РФ;

«2» - Налоговый НЕРЕЗИДЕНТ РФ;

«3» - Иностранец - высококвалифицированный специалист;

«4» - Иностранец-участник государственной программы добровольного переселения соотечественников, проживающих за рубежом;

«5» - Иностранец, имеющий статус беженца или получивший временное убежище на территории РФ;

«6» - Иностранец, осуществляющий трудовую деятельность на основании патента.

•В  таблицу [ЗП налоги] добавлено поле [Статус налогоплательщика НДФЛ] - ссылка на Классификатор.

•Статус налогоплательщика теперь указывается в Личной карточке сотрудника на закладке Налоги - ИМНС/ПФР в новом поле Статус налогоплательщика НДФЛ. Если это поле не заполнено, то признак Статус налогоплательщика в Справке о доходах заполняется, как и раньше: 1, если на закладке Налоги в поле Удерживать подоходный проставлено Резидент, и 2, если на закладке Налоги в поле Удерживать подоходный проставлено Нерезидент. При этом при составлении справки о доходах берется последнее значение на дату конца периода составления Справки. В рамках этого изменения добавлена функция эталона [Сотрудник.ЗП Статус налогоплательщика НДФЛ].

•Таблица [ГНИ код документ] пополнена новыми кодами (исправлена процедура [з/п Заполнение ГНИ код документ]):

15 - Разрешение на временное проживание в Российской Федерации

19 - Свидетельство о предоставлении временного убежища на территории Российской Федерации

23 - Свидетельство о рождении, выданное уполномоченным органом иностранного государства

24 - Удостоверение личности военнослужащего Российской Федерации

•Добавлен  новый Тип ставки Налог с дивидендов нерезидента и ставка 15% с 01.01.2015. Теперь подоходный налог с доходов нерезидентов рассчитывается по этой ставке, если она заполнена, если нет - то как и в прежних версиях, по ставке из таблицы [Ставка подоходного]. В рамках этой модернизации  добавлена процедура эталона [ЗП.Ставка подоходного с дивидендов нерезидента], исправлены  процедуры [з/п удержание/Подоходный налог], [Заполнение Ставка по дате].

 

Указанные выше изменения будут доступны при синхронизации БД эталоном текущей версии.

•Модернизирована выгрузка Декларации по НДС: добавлена выгрузка Раздела 12, содержащего сведения по выставленным счетам –фактурам  в соответствии с п.5 статьи 173 НК. Под указанные счета-фактуры должен быть отведен отдельный вид документа.  

На форму [Стандартная. Декларация по НДС 2015. Экспорт в XML] добавлен новый параметр под заголовком Раздел 12. При его выставлении  появляется дополнительный параметр Вид документа, где должен быть указан вид документа для счетов- фактур, относящихся к п.5 статьи 173 НК.  При выставленных параметрах для Раздела 12 появляется  дополнительная закладка Выставл. с/ф, на которой после нажатия кнопки <Выборка> отображаются данные подлежащие выгрузке. Информация в этом разделе формируется по следующим реквизитам документов:

- № с/ф – поле Документ.[Номер счета-фактуры], если это поле не заполнено, то поле Документ.[Номер];

- Дата с/ф – поле Документ.[Дата счета-фактуры], если это поле не заполнено, то  поле Документ.[Дата]; по  этой же дате  происходит отбор документов для указанного периода;

- Наименование покупателя – поле Документ.[Получатель];

- Стоимость по с/ф всего – поле Документ.[Сумма спецификации всего];

- в т.ч. сумма без НДС - поле Документ.[Сумма без налогов];

- в т.ч. сумма НДС - сумма значений полей Документ.[Сумма НДС 10], Документ.[Сумма НДС 18], Документ.[Сумма НДС 20].

При выгрузке данных Раздела 12 создается отдельный файл XML  с префиксом NO_NDS.12.
