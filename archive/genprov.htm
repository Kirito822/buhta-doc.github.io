Настройка фильтра по контурам учета.

 

В релизе 82 от 16.05.08 расширен пользовательский интерфейс для настройки фильтра по контурам учета:

 

•добавлены следующие флаги:

- Генерация проводок по контурам учета - генерация проводок, у которых счет и корр. счет принадлежат одному и тому же контору учета, отмеченному в списке контуров;

- Генерация проводок с пустым корр. счетом – генерация проводок, у которых один счет принадлежит контуру учета, отмеченному в списке контуров, а другой счет не заполнен.

- Генерация проводок с разными контурами – генерация проводок, у которых хотя бы один из счетов принадлежит контуру учета, отмеченному в списке контуров.

 

•список контуров дополнен двумя значениями:

- <Пустой контур>

- <Пустой счет>.

 

Возможные варианты настроек.

1.Значение по умолчанию (при входе в режим "Перегенерация проводок в документах") - флаг "Генерация проводок по контурам учета" снят, остальные настройки для работы с контурами учета недоступны. В этом случае перегенерация производится по всем проводкам отмеченных видов документов.
Аналогичный результат достигается при установке флагов "Генерация проводок по контурам учета" и "Генерация проводок с разными контурами" и выборе всех значений из списка контуров, включая значения <Пустой контур> и <Пустой счет>.
2.Установлен флаг "Генерация проводок по контурам учета", два остальных сняты. Перегенерация производится по всем проводкам, у которых счет и корр. счет принадлежат одному контуру, выбранному в списке контуров. Это относится и к проводке, у которой счет и корр. счет не заполнен. Для этого в списке контуров должно быть выбрано значение <Пустой счет>.
3.Установлены флаги "Генерация проводок по контурам учета" и "Генерация проводок с пустым корр. счетом" (этот параметр доступен, если в списке выбранных контуров нет значения <Пустой счет>). Снят флаг "Генерация проводок с разными контурами". В этом случае производится перегенерация проводок, у которых счет и корр. счет принадлежат одному контуру, выбранному в списке контуров, или у проводки имеется только счет с контуром указанным в списке, а корр. счет не заполнен.
4.Установлены флаги "Генерация проводок по контурам учета" и "Генерация проводок с разными контурами". Снят флаг "Генерация проводок с пустым корр. счетом". В списке выбранных контуров возможно любое сочетание значений. Производится перегенерация проводок, у которых хотя бы один из счетов принадлежит выбранному контуру.
5.Установлены все три флага: "Генерация проводок по контурам учета", "Генерация проводок с пустым корр. счетом" (этот параметр доступен, если в списке выбранных контуров, нет значения <Пустой счет>), "Генерация проводок с разными контурами". Производится перегенерация проводок, у которых хотя бы один из счетов принадлежит выбранному контуру, в том числе рассматриваются проводки, у которых только счет с контуром указанным в списке, а корр. счет не заполнен.
