Отчеты

 

[ * ] [20122]

Исправлена Налоговая декларация по налогу, уплачиваемому в связи с применением упрощенной системы налогообложения (меню Отчеты - Формы налоговой отчетности - Налоговая декларация по УСН).

1. исправлены штрих-коды

2. для отчетов, начиная с 2014 года, вместо ОКАТО печатается ОКТМО

3. исправлена выгрузка отчета в файл формата XML.

 

Все объекты, измененные в рамках этой модернизации, входят в состав конфигурации BuhtaForm.mdb.        

 

[ + ] [20120]

Добавлена возможность для составления Унифицированной формы ИНВ-22. Режим вызывается из пользовательской формы [Стандартная. ИНВ-22]. Объекты необходимые для составления указанного документа входят в состав стандартной конфигурации BuhtaUniforme.mdb

 

[ - ] [20151]

Исправлена ошибка построения Сводного отчета, которая проявлялась после изменения иерархии строк отчета в режиме настройки без пересоставления запроса (вызывается через кнопку с изображением треугольника в нижней части окна построения Сводного отчета).

 

Настройка

 

[ + ] [20063]

В форму редактирования запроса добавлена колонка Отключить (становится видимой вместе с  колонкой SQL Select). При установке в этой колонке галочки отключается вычисление поля с типом SQL_Число, SQL_Строка, SQL_Дата, SQL_Текст или SQL_ДаНет - выражение в колонке SQL Select заключается в символы комментария, а значение поля устанавливается равным нулю для полей с типом SQL_Число, SQL_Дата, SQL_ДаНет и пустой строке для полей с типом SQL_Строка и SQL_Текст. При снятии  галочки символы комментария удаляются, и восстанавливается прежнее вычисляемое выражение.

 

Зарплата См. также Конфигурация ''Стандартная_Зарплата_...''

 

[ * ] [20127]

Исправлены формы [Стандартная. Архивный отчет в эл.виде (Дополнения)] и [Стандартная. Архивный отчет в электронном виде] входящие в стандартную конфигурацию BuhtaForm.mdb - добавлена выгрузка отчета СВЕДЕНИЯ О ЧИСЛЕННОСТИ И ЗАРАБОТНОЙ ПЛАТЕ РАБОТНИКОВ (форма статистического наблюдения П-4). Отчет составляется через пользовательскую форму [З/П Печать Сведений о численности-ЗП П-4. 2013г] по шаблону [Сведения о численности-ЗП П-4.Форма 2013г].

 

[ * ] [20148]

В связи с изменением алгоритма сбора данных о начисленных и уплаченных взносах на ОПС и экспорта данных в XML-файлы в формате 2014 года (см.Конфигурация ''Стандартная_Зарплата_...'' ) внесены изменения в структуру БД - в таблицу [ПФР Начисления] добавлены поля [ОМС Доход], [ОМС Обл.доход], [ОМС взнос], [ОМС превыш.доход], [ОПС превыш.взнос], [Доп.взнос1], [Доп.взнос2]. Требуется синхронизация БД эталоном текущей версии.

 

Информация для разработчиков

 

[ * ] [20099]

Для шаблонов FastReport 2 (тип шаблона=0) и FastReport 4 (тип шаблона=4) модифицирована функция DIGPROPIS.

В функцию DIGPROPIS добавлен второй (необязательный) параметр <До знака>

    <До знака> >= 0 - с округлением дробной части (по умолчанию до двух знаков),

    <До знака> = -1 - без округления дробной части
