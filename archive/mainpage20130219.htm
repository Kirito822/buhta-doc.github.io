Прайс-лист

 

[ * ] [19183]

В форму настройки Прайса (меню Данные - Настройка - Прайс-лист) на закладку Дополнительно добавлен флаг Заполнять количество только в Форме запроса количества. Если он выставлен, то ввод количества при выборе товара выполняется только через форму запроса количества, при этом ввод количества другим способом запрещен.

 

Справочники

 

[ * ] [19160], [19246]

В режим копирования ТМЦ (пункт контекстного меню списка ТМЦ Копировать или "горячие" клавиши Ctrl+C) добавлена возможность при копировании ТМЦ вместе с комплектом - основным или дополнительными, устанавливать дату начала действия комплекта.

В форму копирования добавлен флаг Установить дату начала действия и поле для ввода даты, которые доступны в режиме копирования комплектов.

В том случае если флаг выставлен:

- копируются действующие на текущую дату комплекты - основной и дополнительные,

- дата начала действия устанавливается из формы,

- дата окончания действия комплектов устанавливается = 0.

Если режим копирования комплектов производится при снятом флаге Установить дату начала действия, то копирование производится для всех указанных комплектов, и дата начала и окончания действия остаются неизменными.

Настройка

 

[ + ] [19236]

Для инициализации имени сервера и номера порта при запуске программы BuhtaClient.exe разработан файл BuhtaClient.ini следующей структуры:

 

[MOBILEPARAM]

SERVERNAME=

SERVERPORT=

 

[NOMOBILEPARAM]

SERVERNAME=

SERVERPORT=

 

При запуске BuhtaClient.exe, анализируется наличие файла BuhtaClient.ini. Если файл существует, то для версии запущенной с параметром mobile считывается настройка из секции [MOBILEPARAM], без параметра mobile - из секции [NOMOBILEPARAM].

- Если в ini-файле в соответствующей секции указаны значения SERVERNAME и SERVERPORT, то именно они подставляются в поля Сервер и Порт в окне входа в систему Бухта.

- Если в ini-файле в соответствующей секции указано значение SERVERNAME, но не указано значение SERVERPORT, то при запуске BuhtaClient.exe имя сервера берется из ini-файла, для порта устанавливается значение 7070.

- Если в ini-файле в соответствующей секции не указано значение SERVERNAME, то независимо от того, указано или нет значение параметра SERVERPORT, имя сервера и номер порта считываются из реестра;  если значение в реестре не указано, то  имя сервера в поле Сервер вводится пользователем вручную, для порта устанавливается значение 7070.

 

Информация для разработчиков

 

[ + ] [19230]

В дизайнер шаблонов типа FastReport4 добавлен объект TfrxQRcode, предназначенный для вывода на печать штрих-кода типа QR.

Рекомендуемые значения свойств AutoSize=true, Stretched=false.

Для вывода на печать штрих-кода необходимо заполнить или поле Text, или, если штрих-код берется из базы данных, то указать DataSet и DataField. Поле базы данных должно быть текстовым.

 

[ * ] [19294]

Для события ApplicationOnAfterSaveXLSDocument, которое возникает после экспорта стандартных списков в Excel, добавлены 2 новых параметра:

- AViewName - содержит имя запроса, по которому построен список,

- RootTableName - содержит имя основной таблицы из запроса в параметре AViewName.
