Справочники

 

[ - ] [16777]

Исправлена ошибка, в результате которой было невозможно в списке видов иерархического справочника (с настройкой иерархии по виду) снять или поставить мышкой флаг для пометки строки у родительского узла нераскрытой ветви.

 

Курсы валют

 

[ * ] [17027]

Модифицирован режим Данные - Курсы валют (изменены формы [Стандартная. Курсы валют] и [Импорт курсов валют (РБК)] входящие в состав конфигурации BuhtaConfig).

На форму [Стандартная. Курсы валют] добавлен флаг Оповещать сотрудников о загрузке/изменении курсов. При установленном флаге с помощью кнопки <Список оповещения> можно отметить пользователей, кому следует послать сообщение об изменении курса валют. Флаг Фильтр по группам и кнопка <Группы пользователей> служат для сокращения просматриваемого списка пользователей.

Для того чтобы пользователю не надо было после получения сообщения перезагружать программу для ввода в действие обновленного курса, можно добавить обработку события OnGetMessage в скрипт ApplicationEvent. См. раздел Информация для разработчиков.

 

Генерация проводок

 

[ - ] [17048]

В режиме Изменение параметров документов (вызывается из списка документов по контекстному меню Сервис - Изменение параметров документов или нажатием клавиши F6) при смене вида документов с генерацией проводок (выставленные флаги Смена вида,   Сменить признак "проведен" на,   Да/Нет,   Произвести генерацию проводок)  отрабатывала процедура генерации исходного вида. Ошибка исправлена.

 

Настройка

 

[ * ] [17031]

Добавлена обработка локальной настройки Использовать кнопку "Выборка" для просмотра справочников (меню Настройка - Локальные настройки, закладка Справочники) для справочников ОС и НМА и Сотрудники, для случая, когда согласно настройке иерархии эти справочники отображаются в окне, состоящем из двух частей - в виде иерархического списка и расшифровки текущей записи этого списке.

В 107 релизе подобная модернизация была выполнена для справочника ТМЦ. См. раздел Релиз 107 - Настройка - Заявка [16933]

 

[ * ] [17059]

Для отчетов, составленных в формате FastReport версии 4, не обрабатывалась настройка Запрос печати в архив из общих настроек системы (меню Настройка - Общая настройка), а именно при установленном значении этой настройки Всегда сохранять в окне предварительного просмотра отчета не проставлялся флаг в поле В архив. Недочет устранен.

 

Сервис

 

[ - ] [16983]

В режиме экспорта/импорта через Конфигуратор для таблиц Вид остатка и Настройка полей добавлены поля, ранее не подлежавшие экспорту/импорту. Информация для разработчиков

 

[ * ] [17027]

Для того чтобы произошло обновление курсов валют сразу же после получения сообщения "Загружены новые курсы валют.", нужно в скрипт ApplicationEvent, в обработку события OnGetMessage добавить следующий текст:

 

Sub ApplicationOnGetMessage(ByRef ShowDlgForm, ByRef FromUser, ByRef InToUser, ByRef MessageStr) ''

if MessageStr = "Загружены новые курсы валют." then

IsAppLoad = True  

KursLoad

IsAppLoad = False

end if

End Sub
