[ * ] [21769]

Внесены следующие изменения в режим составления платежных поручений:

•Добавлена возможность указания произвольного периода в рамках расчетного месяца, за который  собираются данные для создания платежного поручения. На форму [Стандартная. Составление платежных поручений] (входит в состав стандартной конфигурации BuhtaConfig) добавлен новый параметр настройки – признак За период, меньше расчетного месяца. Если он выставлен, то появляется поле, в котором следует указать период в рамках значения из поля настройки Расчетный месяц.

•Модифицирован алгоритм заполнения платежного поручения с типом По запросу:

1.В карточке настройки платежного поручения с типом По запросу для параметра Указать в поле ПП значение группировки разрешено выбирать целые и ссылочные пользовательские поля таблицы Документ.
2.В настройках платежного поручения у поля Вид докспец добавлен признак "из запроса". Если он выставлен, следует выбрать нужное поле из указанного в настройках запроса, в котором находится значение Вида докспец. Подобная настройка позволит создавать спецификации документа разного вида в одном документе платежного поручения. Она может быть использована для выделения платежей по больничным / отпускным и НДФЛ, если они начислены в одном расчетном листке. В рамках данной модификации в эталон текущей версии добавлены новые поля в таблицу [ЗП перечисление]:  [ПП Запрос Вид докспец Ok] и [ПП Запрос Вид докспец].
