Справочники

 

[ * ] [17659]

Обновлен список кодов тарифов плательщика С/В для 2011г (меню Справочники - Прочие справочники - Классификатор, фильтр по виду Код тарифа плательщика С/В). Настройка по юр.лицу производится в режиме Справочники - Настройка - Организации - Юридические лица, карточка редактирования Юр.лица, закладка Отчетность в фонды, параметр Код тарифа плательщика С/В.

 

Отчеты

 

[ * ] [17548]

•В соответствии с Приказом ФНС РФ от 15.12.2010 №ММВ-7-3.730@ добавлена новая форма Декларации по налогу на прибыль (меню Отчеты - Формы налоговой отчетности - Налоговая декларация по налогу на прибыль). При составлении отчета за 2010 год или за любой период 2011 года и далее вызывается новый шаблон отчета [Стандартный. Нал. декл.по налогу на прибыль 2011].

•Изменена форма [Стандартная. Архивный отчет в электронном виде], отвечающая за выгрузку архивных отчетов - добавлена возможность выгрузки отчета, составленного по шаблону [Стандартный. Нал. декл.по налогу на прибыль 2011].

 

Все добавленные и измененные объекты входят в состав стандартной конфигурации BuhtaForm.

 

Генерация проводок

 

[ + ] [17549]

В режим настройки генерации проводок (меню Настройка - Генерация проводок) для алгоритмов списания по FIFO добавлена возможность указания сортировок:

+Договор.[Дата сч.-фактуры],+Договор.Номер

-Договор.[Дата сч.-фактуры],-Договор.Номер

 

Зарплата См. также раздел Конфигурация ''Стандартная_Зарплата_...''

 

[ * ] [17538]

Модернизирован режим сбора данных о начисленных и уплаченных взносах на ОПС в формате 2011 года (вызывается через пользовательскую форму [З/П. Экспорт в XML-файл для ПФР. Формат 2011г]).

О новых функциональных возможностях см. раздел Конфигурация ''Стандартная_Зарплата_...''.

Помимо добавленных и исправленных объектов, которые входят в состав конфигурации Стандартная_Зарплата_216,

внесены следующие изменения в текущий эталон базы данных (файл BuhtaClient.etl):

1)В таблицу [ПФР Начисления] добавлены поля:

- [ОПС страх] и [ОПС накоп] - для сбора данных о начисленных взносах помесячно,

- [Упл ОПС страх], [Упл ОПС накоп] - для сбора данных об уплаченных взносах помесячно.

2)В таблицу [ПФР Индив.сведения] добавлены поля

- [Упл ОПС страх], [Упл ОПС накоп], [Пред.Упл ОПС страх], [Пред.Упл ОПС накоп] - для сбора данных об уплаченных взносах за текущий и предыдущий период суммарно за весь расчетный период.

3)В табл. [ПФР категория НП]

- добавлены поля [Дата начала], [Дата конца]

- увеличена длина поля [Название] с 50 до 200 символов.

4)Изменена процедура эталона [Заполнение ПФР Категория НП].
5)Изменена процедура эталона [з/п Заполнение ГНИ код категория].

 

[ * ] [17580]

Модифицирован отчет по страховым взносам РСВ-1 (пользовательская форма [З/П Расчет по взносам на ОПС и ОМС РСВ-1 ПФР]).

При составлении отчета, начиная с 1 квартала 2011 отчет РСВ-1, вызывается новый шаблон отчета [З/П Расчет по взносам на ОПС и ОМС РСВ-1 ПФР 2011] (входит в состав конфигурации Стандартная_Зарплата_216).

Изменен также формат выгрузки отчета - формы [Стандартная. Архивный отчет в электронном виде],[Стандартная. Архивный отчет в эл.виде (Дополнения)], отвечающие за выгрузку архивных отчетов, входят в состав конфигурации BuhtaForm.

 

[ - ] [17643]

Исправлена процедура эталона [З/П Доходы и дни к расчету среднего] - для алгоритма учета дней Из начислений и статуса (настраивается: меню Зарплата - Настройка - Основная настройка, закладка Отпуск, больничный, средний, параметр Брать дни к расчету среднего) при расчете среднего по календарным дням для отпуска и больничного, не учитывались дни к исключению, если в месяце не было доходов.
