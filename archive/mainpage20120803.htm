На карточку редактирования записи из таблицы ЗП Инфо (вызывается из режима Зарплата - Личные карточки сотрудников, закладка Опер.инф нажатием кнопок <Добавить>, <Изменить>) добавлены кнопки <Сохранить>, <Отмена>, которые были ошибочно удалены в релизе 132.

 

Разное

 

[ + ] [18681]

Реализована привязка файлов к записям таблиц системы БУХта путем перетаскивания файлов (drag & drop) из списочных окон системы Windows.

Можно переносить один или несколько файлов, предварительно их выделив.

Файлы привязываются 2 способами:

- Файлы переносятся в окно со списком записей. Файлы привязываются к записи, на которой стоял курсор в момент начала переноса файла.

- Файлы переносятся в окно Файл (вызывается из списка по "горячим" клавишам Alt+F) или на закладку Файл карточки редактирования (документа, договора).

 

[ - ] [18825]

Исправлено некорректное закрытие окна со списком привязанных файлов, в том случае, когда родительское окно, из которого оно было вызвано, уже закрыто.
