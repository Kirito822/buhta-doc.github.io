статистического наблюдения:

- СВЕДЕНИЯ О ПРОИЗВОДСТВЕ И ОТГРУЗКЕ ТОВАРОВ И УСЛУГ, форма № П-1

- СВЕДЕНИЯ ОБ ОБЪЕМЕ ПЛАТНЫХ УСЛУГ НАСЕЛЕНИЮ ПО ВИДАМ, приложение № 3 к форме № П-1

- СВЕДЕНИЯ О СОСТОЯНИИ РАСЧЕТОВ ЗА ОТГРУЖЕННУЮ ПРОДУКЦИЮ, ВЫПОЛНЕННЫЕ РАБОТЫ (УСЛУГИ), форма № 1-РП

- СВЕДЕНИЯ О ФИНАНСОВОМ СОСТОЯНИИ ОРГАНИЗАЦИИ, форма № П-3

- ОСНОВНЫЕ СВЕДЕНИЯ О ДЕЯТЕЛЬНОСТИ ОРГАНИЗАЦИИ, форма  № 1-предприятие;

- СВЕДЕНИЯ О ЗАТРАТАХ НА ПРОИЗВОДСТВО И ПРОДАЖУ ПРОДУКЦИИ (ТОВАРОВ, РАБОТ, УСЛУГ), форма № 5-З;

- СВЕДЕНИЯ ОБ ОСНОВНЫХ ПОКАЗАТЕЛЯХ ДЕЯТЕЛЬНОСТИ МАЛОГО ПРЕДПРИЯТИЯ, форма № ПМ.

 

и бухгалтерских форм:

- БУХГАЛТЕРСКИЙ БАЛАНС - вызывается из пользовательской формы [Стандартная. Бухгалтерский баланс_91],

- ОТЧЕТ О ПРИБЫЛЯХ И УБЫТКАХ - вызывается из пользовательской формы [Стандартная. Отчет о прибылях и убытках_91],

- ОТЧЕТ О ДВИЖЕНИИ ДЕНЕЖНЫХ СРЕДСТВ - вызывается из пользовательской формы [Стандартная. Отчет о движении денежных средств_91].

 

О  проектирование запросов для составления бухгалтерской и статистической отчетности, а также

получении и отображении в отчетах данных из архивных отчетов см. раздел справки Настройка бухгалтерской и статистической отчетности.

 

Данные всех вышеуказанных отчетов могут быть сохранены в таблицу _Показатели из архивных отчетов для дальнейшего использования.

Если отчет сохранен в таблице _Показатели из архивных отчетов, его невозможно удалить из списка архивных отчетов. В этом случае, следует через контекстное меню списка архивных отчетов, пункт Пользовательские режимы, вызвать пользовательскую форму [Стандартная. Удаление показателей отчета], которая позволяет удалить данные из таблицы _Показатели из архивных отчетов, после чего возможно удаление этого отчета из списка архивных отчетов.

В соответствии с ПРИКАЗом Росстата от 27.11.2008 N 293 добавлен новый отчет СВЕДЕНИЯ О ДВИЖЕНИИ ЦЕННЫХ БУМАГ И ДОХОДАХ ПО НИМ ПО ИНСТИТУЦИОНАЛЬНЫМ СЕКТОРАМ - Форма № 1-ЦБ.

Отчет вызывается из пользовательской формы [Стандартная. Сведения о движении ЦБ и доходах 1-ЦБ].

Отчет относится к группе отчетов, настройка которых основана на показателях. См. раздел справки Настройка бухгалтерской и статистической отчетности.

 

Пример настроек этого отчета, как и всех отчетов этой группы, содержится в файле Пример_настройки_ФО.mdb  в каталоге Дополнения.

Для импорта/экспорта этого файла в таблицу/ из таблицы [_Настройка форм отчетности] имеются пользовательские формы Импорт настроек ФО и Экспорт настроек ФО.

 

Необходимые для составления отчета объекты входят в конфигурацию BuhtaForm.mdb.

Архивный отчет в электронном виде_91], которая вызывается из списка архивных отчетов (меню Отчеты - Архив составленных отчетов) по нажатию клавиш Alt+R.

Форма [Стандартная. Архивный отчет в электронном виде_91] является модернизацией формы [Стандартная. Архивный отчет в электронном виде]. Изменения состоят в следующем:

1. на экране вместо признака для выбора формата добавлена радио-группа - следует выбирать значение XML, Приказ ФНС РФ от 01.04.2009г № ММ-7-6/228@ (по умолчанию это значение проставляется при указании  даты составления больше 2009 года);

2. добавлена выгрузка в XML отчетов, составленных по шаблонам [Стандартный. Отчет о прибылях и убытках_91], [Стандартный. Отчет о движении денежных средств_91], [Стандартный. Бухгалтерский баланс_91]. Эти отчеты являются модернизацией уже имеющихся отчетов из меню Отчеты - Формы бухгалтерской отчетности и вызываются из пользовательских форм. См. также изменения по заявкам [15577], [15710].

 

[ + ] [15512]

В соответствии с ПРИКАЗом от 12 февраля 2009 г. N 25 добавлен новый отчет СВЕДЕНИЯ О РАСПРЕДЕЛЕНИИ ЧИСЛЕННОСТИ РАБОТНИКОВ ПО РАЗМЕРАМ ЗАРАБОТНОЙ ПЛАТЫ - Форма № 1.Отчет вызывается из пользовательской формы [Стандартный. Сведения о численности по з/п Форма1].

Для составления отчета следует:

- указать отчетный месяц (апрель), шифры для сбора данных о з/п и шифры для сбора данных о ФОТ, а также выбрать способ (Из начислений или Из статуса) для исключения сотрудников из расчета (отсутствовавших по болезни или уходу за больными; находящихся в отпусках по беременности и родам, в дополнительных отпусках по уходу за ребенком, в отпусках без сохранения заработной платы более 5 рабочих дней апреля; не работавших по причине простоя по вине работодателя или по причинам, не зависящим от работодателя и работника, более 5 рабочих дней апреля), и указать номера шифров или типы ЗП статусов

- по кнопке Расчет численности и ФОТ рассчитать ср.численность и ФОТ

 

По кнопке Составить отчет  производится расчет з/п и кол-ва.

Берутся сотрудники, удовлетворяющие условиям:

- ЗП статусом на начало месяца Прием на работу (т.е. постоянный),

- в течение месяца нет статусов Уволен, Принят, Внешний совместитель, По договору.

- нет ни одного шифра или зп статуса, указанного на форме в поле хотя бы 1 день, период кот.пересекается с отчетным месяцем

- кол-во дней пересечения отчетного месяца и шифра или зп статуса, указанного на форме в поле не более 5 раб. дней, не превышает 5 раб.дней по Календарю из ЗП Статуса сотрудника на конец месяца.

Суммы берутся по шифрам, указанным в поле Шифры з/п для расчета размера з/п, если в настройке шифра поле Периодичность = Разовое, то вся сумма, если =Ежеквартальное, то 1/3 суммы, иначе - сумма не попадает.

 

Необходимые для составления отчета объекты:

шаблон [Стандартный. Сведения о численности по з/п Форма1], запросы - [Стандартный. Юр.лицо для Форма1] и [Стандартный. Форма Форма1/1] со скриптом и форма [Стандартный. Сведения о численности по з/п Форма1] входят в конфигурацию BuhtaForm.mdb.

 

[ * ] [15519], [15560]

Внесены следующие изменения в режим составления Кассовой книги (меню Отчеты - Денежные средства - Кассовая книга):

- Исправлена ошибка в алгоритме вывода итоговых листов. Ошибка заключалась в том, что итоговые листы выводились в количестве равном количеству дней отображенных в кассовой книге.

- На форму задания параметров в раздел Нумерация и итоги добавлен флаг Выводить итоги (печатать итоговые листы). Флаг отображается, когда период нумерации итогов не совпадает с периодом составления книги. Например: в разделе Нумерация и итоги указан период Квартал, а в разделе Общие в поле Период - первый или второй месяц квартала. В случае совпадения периодов, итоговые листы печатаются автоматически. Для правильного подсчета номеров листов не рекомендуется устанавливать этот флаг. Флаг Выводить итоги добавлен  для случаев, когда итоговый лист нужен.

Справочники

 

[ * ] [15521]

Для возможности использования в отчетах по ЕСН (например, в отчетах, которые составляются из пользовательской формы [ЕСН. Аванс.отчеты и Декларация по ЕСН и ОПС])  кодов налогоплательщиков ЕСН и ОПС  в таблицу Классификатор добавлены два новых вида классификатора: Код налогоплательщика ЕСН, Код налогоплательщика ОПС.

Справочник Кодов налогоплательщика ЕСН и ОПС импортируется в базу данных при ее синхронизации с новым эталоном. Пользователю после обновления структуры базы следует завести значения для новых полей Код налогоплательщика ЕСН и Код налогоплательщика ОПС в карточку редактирования юр.лиц, закладка Отчетность в фонды (вызывается из меню Справочники - Настройка - Организации - Юридические лица).
