Начиная с 2011 года действуют новые шкалы пониженных ставок страховых взносов для отдельных категорий плательщиков, в связи с чем внесены следующие изменения:

 

•В карточку настройки зарплаты (меню Зарплата - Настройка - Основная настройка) на закладку Страх.взносы добавлена группа параметров Применять пониженные тарифы страховых взносов, для выбора одного из представленных ниже значений:

- Нет

- С/Х производитель, плательщик ЕСХН, организация инвалидов

- Упрощенная система налогообложения (по отдельным ОКВЭД)

- Производство, издание средств массовой информации

- Резидент ТВОЭЗ, разработка информационных технологий

- Участник проекта "Сколково"

Параметры становятся видны, если на закладке Зарплата в поле Расчетный месяц ЗП установлено значение не ранее января 2011 года.

 

•Изменена форма [Стандартная. Настройки ЗП для Юр.лица] (входит в состав Стандартной конфигурации BuhtaConfig) - вызывается по нажатию на кнопку <Настройки для Юр.лица> в карточке настройки зарплаты, предназначена  для настроек параметров, действующих в пределах конкретного юр.лица (если эти настройки отличаются от настроек в основной карточке, для текущего юр.лица берутся именно они).

Если в карточке настройки зарплаты на закладке Зарплата в поле Расчетный месяц ЗП установлено значение не ранее января 2011 года, то на форме Настройки ЗП для Юр.лица появляется группа параметров для выбора вида пониженного тарифа уплаты страховых взносов, аналогичная параметрам на закладке Страх.взносы основной карточки настройки.

 

•При запуске приложения текущей версии, в таблицу Настройка в секцию Зарплата автоматически добавляется новый параметр Пониженный тариф страховых взносов (в случае его отсутствия). Если параметр Сельхоз.шкала ЕСН секции Зарплата равен нулю, то параметр Пониженный тариф страховых взносов приравнивается к нулю,  что соответствует выбору позиции Нет настройки Применять пониженные тарифы страховых взносов в карточке настройки зарплаты.  Если параметр Сельхоз.шкала ЕСН равен 1, параметр Пониженный тариф страховых взносов также приравнивается к 1, что соответствует выбору позиции С/Х производитель, плательщик ЕСХН, организация инвалидов настройки Применять пониженные тарифы страховых взносов в карточке настройки зарплаты.

Для секций Зарплата_ЮЛ<№> таблицы Настройка параметр Пониженный тариф страховых взносов автоматически не создается. Если для конкретного Юр.лица существуют индивидуальные настройки, требуется после  установки новой версии приложения и синхронизации базы данных соответствующим эталоном зайти в настройки зарплаты для Юр.лица, установить нужное значение и нажать на кнопку <Cохранить>.

 

•Исправлены процедуры [ЕСН Заполнение шкалы] и [ЕСН расчет налога с 2010] - обновляются при синхронизации базы данных эталоном текущей версии.

Оборотная ведомость

 

[ + ] [17141]

1.Появилась возможность использования в отчете Оборотная ведомость произвольных (в т.ч. пользовательских) полей таблиц, в дополнение к тем, которые жестко определены в системе.

Для этой цели в общей настройке (меню Настройка - Общая настройка) на закладку Оборотная ведомость добавлена кнопка Дополнительные поля. Кнопка доступна для пользователей, имеющих полный доступ к режиму Сервис / Настройка программы / Инструменты / SQL редактирование.

При нажатии кнопки открывается список (иерархический), содержащий дополнительные поля, которые можно использовать в оборотных ведомостях. Записи можно добавлять, изменять, удалять.

При добавлении поля предлагается выбрать в качестве образца уже имеющееся поле, доступное при настройке оборотки и далее "вручную" изменить его характеристики: имя, под которым оно будет использоваться (FieldName), поле-источник из базы данных (SelectSQL) и необходимые SQL-соединения (JoinSQL).

Добавленные (измененные) поля после перезапуска программы смогут использовать другие пользователи.

При удалении производится проверка, используется ли удаляемое поле в уже настроенных отчетах-оборотках, если да - удаление запрещено.

 

2.Модернизирована карточка настройки Оборотной ведомости  (меню Отчеты - Оборотная ведомость, кнопки добавления, изменения). Ранее для полей, отображаемых в колонке Счет/Субсчет/Субконто в том случае, если они принадлежали к одной таблице, было исключено иерархическое отображение, поля выводились в одну строку через запятую. Теперь предоставлено право выбора режима отображения. Для этого в форму настройки оборотной ведомости добавлена колонка Нов. уровень. При установке галочки в этой колонке поле будет отображаться на следующем уровне, при отсутствии - на том же, через запятую (как раньше). Вышесказанное относится к полям одной таблицы, поле из другой таблицы всегда переносится на следующую строку, независимо от значения колонки Нов. уровень.
