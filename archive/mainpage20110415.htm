В форму [Стандартная.Пересчет] (запускается из списка спецификаций документов и договоров) добавлен вариант расчета пропорционально Количеству 2 для параметров Наценка суммой и Итого.

 

Прайс-лист

 

[ * ] [17623]

Модернизирован режим загрузки Прайс-листа (меню Данные - Прайс-лист):

1) Разработан вариант загрузки без видов ТМЦ (общий список). В настройке Прайс-листа (меню Данные - Настройка - Прайс-лист) следует установить значение Нет у параметра Поле группировки.

2) При использовании поиска по "локатору" применяется соответствующая сортировка списка. Если группировка использует поле Вид, то сортировка - в пределах вида.

3) Для алгоритмов поиска ТМЦ (параметр Тип быстрого поиска в настройках Прайс-листа) добавлен новый вариант Локатор по номеру и названию. Поиск выполняется по полю Номер и название (за вычетом [], по первым 20 символам).

 

Книга покупок и книга продаж

 

[ + ] [17616]

Модифицированы режимы составления Журнала полученных счетов-фактур (меню Документы - Книга покупок, кнопка <Журнал>) и Журнала выставленных счетов-фактур (меню Документы - Книга продаж, кнопка <Журнал>). В экранную форму составления журналов добавлены поля для указания шаблона печати (в список выводятся шаблоны, в названии которых присутствует подстрока Журнал полученных счетов-фактур - для журнала покупок и подстрока Журнал выставленных счетов-фактур - для журнала продаж) и кнопка для перехода к редактированию шаблона.

 

Справочники

 

[ * ] [17580]

По сравнению со 116 релизом комплекса Бухта обновлен список кодов тарифов плательщика С/В для 2011г (меню Справочники - Прочие справочники - Классификатор, фильтр по виду Код тарифа плательщика С/В). Процедура обновления входит в состав эталона, вследствие чего новые коды тарифов будут доступны после синхронизации базы данных эталоном текущей версии.

Настройка кода тарифа плательщика С/В по юр.лицу производится в режиме Справочники - Настройка - Организации - Юридические лица, карточка редактирования Юр.лица, закладка Отчетность в фонды, параметр Код тарифа плательщика С/В.

В следующих пользовательских формах (вызываются из меню Отчеты - Формы налоговой отчетности) для параметра Год в налоговом периоде увеличен диапазон выбираемых значений:
