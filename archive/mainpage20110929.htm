Условные обозначения:

Договоры и документы

 

[ + ] [18114]

В режим создания документа из договора добавлен новый вариант Полная спецификация (кроме отгруженных).  Этот режим работает по алгоритму режима Частичная спецификация (кроме отгруженных) в варианте, когда автоматически отмечаются все записи.

 

Прайс-лист

 

[ * ] [18101], [18118]

Модернизирован режим Прайс-лист (меню Данные - Прайс-лист):

• добавлена возможность иерархического отображения по любому полю группировки, а не только по виду ТМЦ.

•оптимизирован режим обновления данных при раскрытии информации по 2-му субконто.

 

Отчеты

 

[ - ] [17950]

Исправлен шаблон отчета [Стандартный. Бухгалтерский баланс 2011], в котором на листе сравнения данных баланса с оборотной ведомостью - лист 2, в строке 8 указывались неверные ссылки.

Шаблон [Стандартный. Бухгалтерский баланс 2011] вызывается из пользовательской формы [Стандартная. Бухгалтерский баланс 2011] (меню Отчеты - Формы бухгалтерской отчетности - Бухгалтерский баланс), если на форме не выставлен флаг Печатать в форме, рекомендованной ФНС  РФ.

Шаблон входит в состав конфигурации BuhtaForm.mdb.

 

[ + ] [18078]

Добавлена возможность для создания шаблонов отчетов Word \ Excel в версии Microsoft Office 2010.

 

Зарплата См. также Конфигурация ''Стандартная_Зарплата_...''

 

[ * ] [18000]

Внесены изменения в следующие алгоритмы расчета з/п:

- [з/п Удержание/Процент от начислений]

- [з/п прочий/Процент от других шифров]

- [з/п начисление/Процент от других начислений]

Теперь для этих алгоритмов учитывается значение коэффициента, установленного на закладке Привязка шифров в карточке редактирования Шифра ЗП.

Изменения вступят в силу при синхронизации базы данных эталоном текущей версии.

 

[ * ] [18016]

При удалении записей из таблицы ЗП перечисление добавлена проверка на присутствие удаляемого значения в основных настройках зарплаты (меню Зарплата - Настройка - Основная настройка).

 

[ * ] [18098]

Модернизирована форма [Стандартная. Составление платежных ведомостей] - вызывается из меню Зарплата - Составление платежных ведомостей, входит в состав конфигурации BuhtaConfig.mdb. На форму добавлен флаг Заполнять Поставщика, по которому открываются поля с типом и значением Поставщика (с типом и значением субконто).  Для типа поставщика Орг открывается поле Расчетный счет. Если флаг Заполнять Поставщика выставлен, то документ заполняется указанными значениями.

Аналогичные изменения внесены в форму [Быстрый ввод аванса]. См. раздел Конфигурация ''Стандартная_Зарплата_...''
