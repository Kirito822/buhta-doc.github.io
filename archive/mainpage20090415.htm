Книга доходов и расходов

 

[ * ] [15350], [15508], [15538]

Внесены изменения в Книгу доходов и расходов (меню Данные - Книга доходов и расходов).

•Добавлены шаблоны [Книга доходов и расходов (раздел I) 2008],  [Книга доходов и расходов (раздел I) 2009], по которым раздел I Книги доходов и расходов печатается с делением по кварталам и выводом промежуточных итогов, сортировка производится по дате включения в книгу, внутри даты - по порядковому номеру.

•В соответствии с Приказом Минфина от 31.12.2008 №154н утверждена новая форма Книги, в связи с чем для печати титульного листа и раздела III Книги добавлены новые шаблоны [Книга доходов и расходов (титульный лист) 2009] и [Книга доходов и расходов (Расчет сум. убытка) 2009].

•В форму настройки Книги (редактирование раздела) добавлен флаг Округление суммы до целых значений. Если он выставлен, то при выводе на экран при составлении/просмотре книги данные округляются до целых значений.

• На форму запроса параметров печати добавлен флаг Округлять суммы до целых значений, позволяющий распечатать Книгу (раздел I и раздел II) с целыми или дробными значениями независимо от текущей настройки Книги (при открытии формы он автоматически устанавливается в соответствии с настройкой). Соответствующие изменения внесены в шаблоны [Книга доходов и расходов (раздел I) 2008],  [Книга доходов и расходов (раздел I) 2009] и [Книга доходов и расходов (раздел II) 2007].

•На форму просмотра Книги ниже кнопки Составить добавлен флаг Автонумерация. При выставленном флаге строки Книги при составлении перенумеровываются, начиная с номера, на единицу больше последнего порядкового номера за предыдущий период.      

 

Отчеты

 

[ * ] [15173],[15460]

Модернизированы отчеты  статистического наблюдения:

 

a) переделаны отчеты, которые вошли в стандартную конфигурацию 91 релиза - в текущем релизе для этих отчетов  исправлена настройка выборки данных:

- СВЕДЕНИЯ О ПРОИЗВОДСТВЕ И ОТГРУЗКЕ ТОВАРОВ И УСЛУГ, форма № П-1

- СВЕДЕНИЯ ОБ ОБЪЕМЕ ПЛАТНЫХ УСЛУГ НАСЕЛЕНИЮ ПО ВИДАМ, приложение № 3 к форме № П-1

- СВЕДЕНИЯ О СОСТОЯНИИ РАСЧЕТОВ ЗА ОТГРУЖЕННУЮ ПРОДУКЦИЮ, ВЫПОЛНЕННЫЕ РАБОТЫ (УСЛУГИ), форма № 1-РП

- СВЕДЕНИЯ О ФИНАНСОВОМ СОСТОЯНИИ ОРГАНИЗАЦИИ, форма № П-3.

 

b) разработаны новые отчеты:

- ОСНОВНЫЕ СВЕДЕНИЯ О ДЕЯТЕЛЬНОСТИ ОРГАНИЗАЦИИ, форма  № 1-предприятие;

- СВЕДЕНИЯ О ЗАТРАТАХ НА ПРОИЗВОДСТВО И ПРОДАЖУ ПРОДУКЦИИ (ТОВАРОВ, РАБОТ, УСЛУГ), форма № 5-З;

- СВЕДЕНИЯ ОБ ОСНОВНЫХ ПОКАЗАТЕЛЯХ ДЕЯТЕЛЬНОСТИ МАЛОГО ПРЕДПРИЯТИЯ, форма № ПМ.

 

Формирование этих отчетов требует обязательной настройки запросов, которая осуществляется через форму [Настройка форм отчетности. Настройка запросов].

Вызов этой формы производится  через кнопку на пользовательских формах, выполняющих составление отчетов.

Во всех отчетах настраивается столько запросов, сколько периодов выборки присутствует в печатной форме - если в отчете используются данные текущего и предыдущего периодов, то в настроечной форме присутствует 2 настройки.  В формах запрашивается только 1 период - текущий. Остальные периоды определяются по умолчанию.

 

Для сохранения настроек отчетов подобного типа в базу добавлена таблица [_Настройка форм отчетности].

По настройке, для каждой ячейки отчета выбираются показатели - например, суммы сальдо или оборотов по части счетов (субсчетов), а также формулы, SQL- числа, SQL – строки и показатели из сохраненных (архивных) отчетов. Последние рекомендуется использовать в настройке предыдущих периодов. Показатели из сохраненных (архивных) отчетов хранятся в таблице [_Показатели из архивных отчетов] и записываются туда после создания или редактирования отчета в том случае, если пользователь при закрытии отчета отвечает утвердительно на вопрос о необходимости их сохранения.

 

Пример настроек отчетов содержатся в файле Пример_настройки_ФО.mdb, в каталоге Дополнения. Для импорта/экспорта этого файла в таблицу/ из таблицы [_Настройка форм отчетности] имеются пользовательские формы Импорт настроек ФО и Экспорт настроек ФО.

 

Краткий перечень необходимых действий для составления новых отчетов:

 

1 ОСНОВНЫЕ СВЕДЕНИЯ О ДЕЯТЕЛЬНОСТИ ОРГАНИЗАЦИИ, форма  № 1-предприятие.

На форме надо настроить запрос, указать период,  указать шифры для расчета ФОТ, нажать кнопку <Расчет численности и ФОТ> для расчета среднесписочной численности и ФОТ за период, а потом нажать кнопку <Составить отчет>.

 

2. СВЕДЕНИЯ О ЗАТРАТАХ НА ПРОИЗВОДСТВО И ПРОДАЖУ ПРОДУКЦИИ (ТОВАРОВ, РАБОТ, УСЛУГ), форма № 5-З.

На форме надо настроить запросы за отчетный период и период предыдущего года, указать период, за который собирается отчет (нарастающим с начала года), автоматически рассчитается соответствующий период пред.года и появится на экране. Нажать кнопку <Составить отчет>.

 

3. СВЕДЕНИЯ ОБ ОСНОВНЫХ ПОКАЗАТЕЛЯХ ДЕЯТЕЛЬНОСТИ МАЛОГО ПРЕДПРИЯТИЯ, форма № ПМ.

 

3.1. На форме надо настроить запросы за отчетный период и период предыдущего года, указать период, за кот.собирается отчет (нарастающим с начала года), автоматически рассчитается соотв.период пред.года и появится на экране.

3.2. Если в запросах не настроены переменные для заполнения в Разделе 1 данных за отчетный период или за период прошлого года, то следует заполнить поля под заголовком Настройка Раздела 1: надо указать шифры для сбора данных о фонде з/п и соц.выплатах, затем нажать кнопку <Расчет численности и выплат за отчетный период> для сбора данных за расчетный период и (или) кнопку <Расчет численности и выплат за период пред.года> - для сбора данных за период прошлого года. В появившейся на экране форме заполнить поля для расчета численности и нажать кнопку <Расчет>, численность и суммы выплат рассчитаются и форма закроется, а на нашей форме появится текстовое окно с результатами расчета.

3.3. Нажать кнопку <Составить отчет> для составления отчета. Надо помнить, что если данные для Раздела 1 рассчитаны в форме и указаны в настроенных запросах, то в отчет попадут данные из отчетов (надо удалить из настройки запросов переменные по Разделу 1, тогда в отчет попадут данные, рассчитанные по кнопкам <Расчет численности и выплат...>).

 

Все добавленные и измененные объекты входят в конфигурацию BuhtaForm.mdb

В том случае если режим активен (виден столбец пометки строк) и отмечена хотя бы одна строка списка, вывод данных на печать (режим Печать списка) и в таблицу Excel (режим Экспорт в Excel) осуществляются по отмеченным строкам.

 

[ - ] [15474]

В режиме составления Сводного отчета (меню Отчеты - Сводный отчет) исправлена ошибка подведения итогов по уровням иерархии. Ошибка проявлялась в том случае, когда колонка вычислялась по формуле, а итоги по уровням иерархии вычислялись как сумма по колонке.

 

[ + ] [15449]

В режим Отчеты - Архив составленных отчетов добавлена возможность контекстного поиска по текстам архивных отчетов. Режим поиска вызывается по нажатию на кнопку <Поиск в архиве отчетов> на форме Архив составленных отчетов или через подменю Сервис контекстного меню списка отчетов, пункт Поиск в архиве отчетов.

После того как пользователем будет введена строка для поиска, вызывается отдельное окно, в которое при завершении режима поиска будет выведен список найденных отчетов. Для досрочного прерывания режима поиска, следует нажать на кнопку <Стоп> в окне поиска. В этом случае будут выведены все найденные к этому времени отчеты.

Замечание 1: поиск выполняется без учета регистра.

Замечание 2:  поиск по отчетам, созданным на основе шаблонов 2-го типа (MS Excel) не работает для выражений, являющихся  результатом вычислений по формулам.

Замечание 3: Поиск по архивным отчетам, созданным на основе таблиц Документ и Договор доступен из карточки редактирования Документа/Договора закладка Печатные формы, подменю Сервис контекстного меню списка отчетов, пункт Поиск в архиве отчетов.

 

[ + ] [15497]

Разработан отчет Реестр застрахованных лиц, за которых перечислены дополнительные страховые взносы на накопительную часть трудовой пенсии и уплачены взносы работодателя, форма ДСВ-3.

Вызов формы составления отчета [Стандартный. Реестр застрахованных лиц ДСВ-3] осуществляется из списка документов вида Платежное поручение (-) по Alt+F12 - вызывается список пользовательских форм, или через контекстное меню, подменю Пользовательские режимы.

Возможен пакетный режим составления отчетов по отмеченным записям. В этом случае составленные отчеты можно просмотреть в режиме Отчеты - Архив составленных отчетов.

Для отчета Реестр застрахованных лиц ДСВ-3 возможна выгрузка в XML-файл. Выгрузка производится в режиме Отчеты - Архив составленных отчетов через форму [Стандартная. Архивный отчет в электронном виде], которая вызывается из списка архивных отчетов по нажатию клавиш Alt+E.

Объекты необходимые для составления этого отчета входят в конфигурацию BuhtaForm.mdb.

 

Оборотная ведомость

 

[ * ] [15478]

В форму создания документа из оборотной ведомости (меню Отчеты - Оборотная ведомость, кнопка Создать док. ) добавлен флаг Выполнить генерацию проводок.

Если флаг Выполнить генерацию проводок выставлен (значение по умолчанию), то при создании документа  происходит генерация проводок, и в документе выставляется флаг Проведен.

Если  флаг Выполнить генерацию проводок снят, то при создании документа генерации проводок не происходит, и флаг Проведен снят.

 

АРМ

Настройка

 

[ + ] [15403]

На форму Локальной настройки (меню Настройка->Локальные настройки) добавлена закладка Общие, которая содержит флаг Упорядочить иконки при входе в программу.
