Режим ''Корректировка спецификаций договоров''

 

Внимание!!! Режим корректировки спецификаций договоров существенно переработан.

Изменения подробно описаны в данном разделе.

 

Режим вызывается из списка  просмотра договоров клавишей "Коррект.догспец".

 

1.Для этого необходимо открыть список договоров.
2.Отметить (клавиша "Пробел") договора для которых будет применяться этот режим. Если договор один, то все равно его нужно отметить (клавиша "Пробел").
3.Нажать на кнопку "Коррект.догспец".
4.В открывшейся карточке произвести необходимые изменения в данных.
5.Нажать клавишу "Выполнить" для сохранения измененных данных или клавишу "Закрыть" для отмены введенных данных.

 

Алгоритм работы:

1. Анализируются спецификации отмеченных договоров на первом субконто.

2. Создается уникальный список из ТМЦ. Для каждой строки ТМЦ отображается список договоров (спецификаций договоров) в которых присутствует этот товар.

 

В полученном списке можно:

•изменять количество (ручной ввод значения в колонке "Новое кол-во"). Если количество ставится нулевое, то такая спецификация удаляется из договора. Количество можно менять как в строке спецификации(для одной спецификации) , так и для всех спецификаций имеющих этот ТМЦ. Для этого количество нужно ставить в строке ТМЦ и оно будет распределено между спецификциями.

•изменять кол-во упаковок (ручной ввод значения в колонке "Новое кол.уп."). Если количество ставится нулевое, то такая спецификация удаляется из договора. Количество можно менять как в строке спецификации(для одной спецификации), так и для всех спецификаций имеющих этот ТМЦ. Для этого количество нужно ставить в строке ТМЦ и оно будет распределено между спецификциями.

•изменять цену. (ручной ввод значения в колонке "Новая цена"). Цена меняется непосредственно для спецификации (курсор стоит на спецификации).

•По кнопке "Замена цены" можно произвести замену цены в соответствии со справочником Базовых цен по выбранному курсу. Замену цены можно производить для отдельной спецификации (курсор стоит на спецификации) или для всех спецификаций в которых встречается этот товар (курсор стоит на товаре).

•По кнопке "Новый товар" происходит замена ТМЦ в указанных спецификациях. Замену товара можно производить для отдельной спецификации (курсор стоит на спецификации) или для всех спецификаций в которых встречается этот товар (курсор стоит на товаре).

 

Внимание!!! Все изменения заносятся в базу при нажатии на кнопку "Выполнить". Значение в поле "кол-во упаковок" всегда округляется до целого арифметически либо в большую сторону взависимости от настройки, см. Настройка->Общая настройка->Вкладка Документы/Договоры поле: Количество упаковок в спецификациях округлять до:........

 

Дополнительные возможности:

•Вызов карточки редактирования договора, кл.F4 (курсор стоит на спецификации)

•Печать списка, кл.F9
