Отчеты

 

[ * ] [21248]

 

•В режим составления отчета Налоговая Декларация по НДС (вызывается из пользовательской формы [Стандартная. Декларация по НДС 2015. Экспорт в XML], кнопка <Новый отчет> на закладке Выгрузка), добавлена возможность выбора запроса, по которому будет составляться декларация. При выборе запроса Стандартный. Форма НДС/1 (добавлен в конфигурацию BuhtaForm начиная с 173 релиза) появляется дополнительная возможность построения предложенного запроса с помощью системы показателей, в данном случае имеющих код формы НДС.  Режим редактирования запроса Стандартный. Форма НДС/1 вызывается  при нажатии  кнопки справа от поля с именем запроса.

 В режиме редактирования для показателей с кодом формы НДС добавлено два новых типа показателей Книга покупок, Книга продаж, которые можно  использовать в случае отбора счетов –фактур по кодам видов операций. При выборе типов показателей Книга покупок, Книга продаж пользователю предлагается заполнить поля Виды операций и Суммы (возможные значения [Сумма 0], [Сумма без НДС 10], [Сумма без НДС 18], [Сумма без НДС 20], [Сумма всего], ], [Сумма НДС 10], ], [Сумма НДС 18] [Сумма НДС 20] или их комбинации). Рекомендуемая настройка для заполнения показателей Раздела 3 включена в файл Пример_настройки_ФО.mdb, который входит в состав поставки версии. Пользователю, уже работавшему с отчетами по запросам с настройкой показателей, можно импортировать эти показатели из предлагаемого файла выбрав код формы НДС, а затем их отредактировать и дополнить по своему усмотрению. Для пользователей , которые не использовали до этого отчеты  с показателями, достаточно будет примера, который содержится в запросе Стандартный. Форма НДС/1. Этот запрос также может быть отредактирован - названия показателей в запросе формируются следующим образом:  sN1_ N2_N3, где  s –идентификатор настраиваемых показателей, N1 – номер раздела отчета , N2 – код строки в разделе, N3 -код столбца в разделе. Так значение показателя с названием  s2_010_3 заносится в 3-й столбец 10 строки  2-го раздела отчета.

•Добавлена  выгрузка Раздела 2 Налоговой Декларации по НДС в XML-файл.

 

Добавленные и измененные в рамках модернизации объекты входят в состав конфигурации BuhtaForm.

 

Зарплата

 

[ *  ] [21352]

В соответствии с Приказом  ФНС России от 10.09.2015 №ММВ-7-11/387@ «Об утверждении кодов видов доходов и вычетов»:

•обновлены таблицы видов доходов и видов вычетов,

•начиная с 01.01.2016 добавлены следующие новые ставки :

- предельный размер базы для ПФР = 796 000,

- предельный размер базы для ФСС = 718 000,

- МРОТ = 6204,

- увеличен стандартный вычет родителям, усыновителям детей-инвалидов до 12000 рублей,

- предельный размер дохода, на который предоставляется вычет, увеличен до 350000 рублей.

В рамках указанной модификации изменены процедуры эталона :

- [Заполнение Ставка по дате],

- [з/п Заполнение таблицы видов дохода],

- [з/п Заполнение таблицы видов вычета]

 

Обновлена Форма 2-НДФЛ (Справка о доходах физического лица). См. раздел Стандартная_Зарплата
