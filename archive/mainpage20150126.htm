WMS. Складская топология

 

[ - ] [20697]

Топология. Исправлена ошибка в режиме редактирования параметров стеллажа.

 

Списки

 

[ * ] [20686]

При работе в терминальной сессии увеличена скорость экспорта в Excel стандартных списков на локальный диск.

 

Отчеты

 

[ * ] [20678]

В соответствии с Приказом ФНС России от 04.07.2014 N ММВ-7-3/352@  внесены изменения в налоговую декларацию по налогу, уплачиваемому в связи с применением упрощенной системы налогообложения (меню Отчеты – Формы налоговой отчетности – Налоговая декларация по УСН). Изменена выгрузка отчета в файлы формата XML.

Добавленные и измененные в рамках текущей модернизации объекты:

- шаблон [Стандартный. Налоговая декларация по УСН 2014],

- запрос [Стандартный. Налоговая декларация по УСН 2014] со скриптом

- форма [Стандартный. Налоговая декларация по УСН],

- форма [Стандартная. Архивный отчет в электронном виде],

- форма [Стандартная. Архивный отчет в эл.виде (Дополнения)]

входят в конфигурацию BuhtaForm.

 

Оборотная ведомость

 

[ * ] [20687]

В режиме Отчеты – Оборотная ведомость устранена возможность "перетаскивания" колонок из одной группы параметров в другую, например из группы Количество в группу Количество 2.

 

ОС и НМА

 

[ - ] [20707]

Исправлена ошибка, в результате которой редактирование записей таблицы Норма износа с нечисловым полем Номер было недоступно.

 

Информация для разработчиков

 

[ * ] [20717]

Добавлена возможность подключения события OnRecordAdded (добавление новой записи) для таблицы Файл. Для срабатывания этого события в режиме работы с таблицей Файл следует использовать пользовательскую форму, на которой нужно расположить компонент TBuhtaGrid и  в качестве View указать "Файл".
