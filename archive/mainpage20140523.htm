Формы

 

[ + ] [20273]

Разработана новая пользовательская форма [Стандартная. Создание доверенности по договору], которая позволяет создавать доверенности по указанным договорам.

Форма вызывается из контекстного меню списка договоров, пункт Пользовательские режимы, или, если для этой формы указано сочетание горячих клавиш, по нажатию этих клавиш.

Настраиваемые параметры формы для передачи из договора в доверенность:

•Дата - варианты настройки: дата договора или задается пользователем.

•Номер - варианты настройки: из договора, устанавливается согласно автонумерации или задается пользователем.

•Срок действия - варианты настройки: задается конкретной датой или количеством дней от даты доверенности.

•Менять местами поставщика и получателя - логическое поле. В случае  выставленного флага значения поставщика и получателя, указанные в договоре, для полей поставщика и плательщика доверенности меняются местами.

Вне зависимости от настройки заполняется поле Потребитель (выбирается из реквизитов текущего юр.лица) и поле РС Потребителя.

•Копировать спецификацию - варианты настройки: Да, Нет, В одну, если Да или В одну, то доступны дополнительные настройки Полностью или Частично. Корректировка спецификации возможна только при создании доверенности для одной записи договора при установленном значении Частично.

 

Форма [Стандартная. Создание доверенности по договору], а также связанные с ней объекты: запросы [Создание доверенности с частичной спец (1 поз.)], [Создание доверенности с частичной спецификацией] с VB-скриптами и процедура [Договор_Доверенность_создание] входят в состав стандартной конфигурации BuhtaConfig.

 

Настройка

 

[ - ] [20275]

При удалении поля из таблицы в режиме Настройка - Инструменты - Схема данных производилась некорректная проверка на наличие удаляемого поля в настройке Журнала событий - запрещалось удалять поля, наименования которых частично совпадали с наименованиями полей, используемых в этой настройке. Ошибка исправлена.
